# 3. Advanced Methods and Functions

Because dictionaries use `key:value` pairs, they have an entirely unique set of methods designed to fetch, update, and manipulate these pairs.

---

## 1. Master Reference Table
Here are all 11 Dictionary Methods and 6 Built-in Functions that operate on Dictionaries.

### Dictionary Methods
| Method | Description |
|---|---|
| `clear()` | Removes all the elements from the dictionary. (Covered in Part 2) |
| `copy()` | Returns a shallow copy of the dictionary. |
| `fromkeys()` | Returns a dictionary with the specified keys and value. |
| `get()` | Returns the value of the specified key. (Covered in Part 1) |
| `items()` | Returns a list containing a tuple for each key value pair. (Covered in Part 1) |
| `keys()` | Returns a list containing the dictionary's keys. (Covered in Part 1) |
| `pop()` | Removes the element with the specified key. (Covered in Part 2) |
| `popitem()` | Removes the last inserted key-value pair. |
| `setdefault()`| Returns the value of the specified key. If the key does not exist: insert the key, with the specified value. |
| `update()` | Updates the dictionary with the specified key-value pairs. (Covered in Part 2) |
| `values()` | Returns a list of all the values in the dictionary. (Covered in Part 1) |

### Built-in Functions for Dictionaries
| Function | Description |
|---|---|
| `len()` | Returns the number of pairs. (Covered in Part 1) |
| `any()` | Returns True if any key is True. (Covered in Part 1) |
| `all()` | Returns True if all keys are True. (Covered in Part 1) |
| `dict()` | Converts a list of tuples into a dictionary. |
| `sorted()` | Returns a brand new sorted list of the keys. |

## 2. Deep Dive: Dictionary Methods

### `fromkeys()`
Creates a new dictionary with the given keys, and sets them all to the exact same value.
**Input:**
```python
keys = ('key1', 'key2', 'key3')
value = 0

new_dict = dict.fromkeys(keys, value)
print(new_dict)
```

**Output:**
```text
{'key1': 0, 'key2': 0, 'key3': 0}
```


### `popitem()`
Removes the absolute last inserted `key:value` pair from the dictionary, and returns it as a tuple.
**Input:**
```python
car = {"brand": "Ford", "model": "Mustang"}
removed = car.popitem()

print("Removed tuple:", removed)
print("Remaining dict:", car)
```

**Output:**
```text
Removed tuple: ('model', 'Mustang')
Remaining dict: {'brand': 'Ford'}
```


### `setdefault()`
Returns the value of a key. However, if the key does NOT exist, it will automatically insert the key and set it to a default value you provide!
**Input:**
```python
car = {"brand": "Ford", "model": "Mustang"}

# Exists! Returns 'Mustang'
x = car.setdefault("model", "Bronco")

# Doesn't exist! Inserts 'color': 'white' and returns 'white'
y = car.setdefault("color", "white")

print(car)
```

**Output:**
```text
{'brand': 'Ford', 'model': 'Mustang', 'color': 'white'}
```


### `copy()`
Like lists, dictionaries share memory unless you explicitly copy them.
**Input:**
```python
original = {"brand": "Ford"}
safe_copy = original.copy()
```

## 3. Deep Dive: Built-in Functions

### The `dict()` Constructor
You can create a dictionary from scratch using the `dict()` built-in constructor instead of using curly braces `{}`. This is incredibly useful when converting a list of tuples into a dictionary!

**Input:**
```python
# A list of tuples containing pairs of data
data_list = [("brand", "Ford"), ("model", "Mustang"), ("year", 1964)]

# Convert the list of tuples directly into a Dictionary!
my_dict = dict(data_list)
print(my_dict)
```

**Output:**
```text
{'brand': 'Ford', 'model': 'Mustang', 'year': 1964}
```


### `sorted()`
The `sorted()` function returns a brand new, sorted **List** of the dictionary's *keys*. It does not sort the values, only the keys!

**Input:**
```python
student_grades = {
    "Zack": 85,
    "Alice": 92,
    "Bob": 78
}

# This will return a list of the names (keys) sorted alphabetically!
sorted_names = sorted(student_grades)
print("Sorted Keys:", sorted_names)
```

**Output:**
```text
Sorted Keys: ['Alice', 'Bob', 'Zack']
```
