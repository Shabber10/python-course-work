# 2. Modifying and Nested

Now that you know how to access data in a dictionary, we will look at how to safely add, modify, and remove data, as well as how to store dictionaries inside of other dictionaries!

---

## 1. Changing and Adding Items
You can change the value of a specific item by referring to its key name. If the key does not exist, Python will automatically add it as a brand new `key:value` pair!

**Input:**
```python
car = {
  "brand": "Ford",
  "model": "Mustang",
  "year": 1964
}

# Changing an existing item
car["year"] = 2020

# Adding a brand new item
car["color"] = "red"

print(car)
```
**Output:**
```text
{'brand': 'Ford', 'model': 'Mustang', 'year': 2020, 'color': 'red'}
```

### The `update()` Method
The `update()` method will update the dictionary with the items from the given argument. If the item does not exist, it will be added. The argument must be a dictionary!

**Input:**
```python
car = {
  "brand": "Ford",
  "model": "Mustang",
  "year": 1964
}

# Update the year, and add a color!
car.update({"year": 2024, "color": "white"})
print(car)
```
**Output:**
```text
{'brand': 'Ford', 'model': 'Mustang', 'year': 2024, 'color': 'white'}
```

---

## 2. Removing Items (`pop`, `del`, `clear`)

- `.pop(key)`: Removes the item with the specified key name.
- `del`: A keyword that removes the item with the specified key name, or deletes the entire dictionary completely.
- `.clear()`: Empties the dictionary completely.

**Input:**
```python
car = {
  "brand": "Ford",
  "model": "Mustang",
  "year": 1964,
  "color": "red"
}

# Pop the model
car.pop("model")
print("After Pop:", car)

# Delete the color using the del keyword
del car["color"]
print("After Del:", car)

# Clear the whole thing
car.clear()
print("After Clear:", car)
```
**Output:**
```text
After Pop: {'brand': 'Ford', 'year': 1964, 'color': 'red'}
After Del: {'brand': 'Ford', 'year': 1964}
After Clear: {}
```

## 3. Nested Dictionaries
A dictionary can contain dictionaries. This is called nested dictionaries, and it is the exact same structure as a JSON file!

**Input:**
```python
# Creating a nested dictionary of families
myfamily = {
  "child1" : {
    "name" : "Emil",
    "year" : 2004
  },
  "child2" : {
    "name" : "Tobias",
    "year" : 2007
  },
  "child3" : {
    "name" : "Linus",
    "year" : 2011
  }
}

# Accessing items in a nested dictionary
print("The whole family:", myfamily)
print("Child 2's name:", myfamily["child2"]["name"])
print("Child 3's birth year:", myfamily["child3"]["year"])
```
**Output:**
```text
The whole family: {'child1': {'name': 'Emil', 'year': 2004}, 'child2': {'name': 'Tobias', 'year': 2007}, 'child3': {'name': 'Linus', 'year': 2011}}
Child 2's name: Tobias
Child 3's birth year: 2011
```
