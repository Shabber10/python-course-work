## 3. The `continue` Statement
With the `continue` statement we can stop the current iteration, and continue with the next.

**Input:**
```python
i = 0
while i < 5:
    i += 1
    if i == 3:
        continue
    print(i)
```
**Output:**
```text
1
2
4
5
```
*(Notice that `3` is missing from the output)*
