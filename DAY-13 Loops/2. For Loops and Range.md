# 2. For Loops and Range

A `for` loop in Python is used for iterating over a sequence (that is either a list, a tuple, a dictionary, a set, or a string). 

This is less like the `for` keyword in other programming languages (like C++ or Java), and works more like an iterator method as found in other object-orientated programming languages.

---

## 1. The `for` Loop Syntax
With the `for` loop we can execute a set of statements, once for each item in a list, tuple, set etc.

**Input:**
```python
fruits = ["apple", "banana", "cherry"]

# Notice that you do not need to define 'x' beforehand!
for x in fruits:
    print(x)
```
**Output:**
```text
apple
banana
cherry
```

### Looping Through a String
Even strings are iterable objects, they contain a sequence of characters!
**Input:**
```python
for letter in "banana":
    print(letter)
```
**Output:**
```text
b
a
n
a
n
a
```

---

## 2. The `range()` Function
To loop through a set of code a specified number of times, we can use the `range()` function. The `range()` function returns a sequence of numbers, starting from 0 by default, and increments by 1 (by default), and ends at a specified number.

**Input:**
```python
# The loop will run 6 times, from 0 to 5. It does NOT include 6!
for x in range(6):
    print(x)
```
**Output:**
```text
0
1
2
3
4
5
```

### Advanced Range (Start, End, Step)
You can specify a start point, an end point, and a step sequence: `range(start, end, step)`.
**Input:**
```python
# Start at 2, end at 10, step by 2
print("Even numbers:")
for x in range(2, 10, 2):
    print(x)
```
**Output:**
```text
Even numbers:
2
4
6
8
```

---

## 3. The `else` in For Loop
The `else` keyword in a `for` loop specifies a block of code to be executed when the loop is totally finished. 

*(Note: The `else` block will NOT be executed if the loop is stopped early by a `break` statement.)*

**Input:**
```python
for x in range(3):
    print(x)
else:
    print("Finally finished!")
```
**Output:**
```text
0
1
2
Finally finished!
```

---

## 4. Nested Loops
A nested loop is a loop inside a loop. The "inner loop" will be executed one full time for *every single iteration* of the "outer loop".

**Input:**
```python
adj = ["red", "big", "tasty"]
fruits = ["apple", "banana", "cherry"]

# For every single adjective, it will loop through ALL THREE fruits!
for x in adj:
    for y in fruits:
        print(x, y)
```
**Output:**
```text
red apple
red banana
red cherry
big apple
big banana
big cherry
tasty apple
tasty banana
tasty cherry
```

---

## 5. Looping with `enumerate()`
Often when looping through a sequence, you need to know the **index** (the current position) as well as the value. `enumerate()` handles this beautifully by returning both!

**Input:**
```python
fruits = ["apple", "banana", "cherry"]

# enumerate gives you (index, value) for each item!
for index, fruit in enumerate(fruits):
    print(f"Index: {index}, Fruit: {fruit}")
```
**Output:**
```text
Index: 0, Fruit: apple
Index: 1, Fruit: banana
Index: 2, Fruit: cherry
```

---

## 6. Looping through Multiple Lists with `zip()`
If you have two or more lists and you want to loop through them at the exact same time in parallel, you can use the `zip()` function.

**Input:**
```python
names = ["Alice", "Bob", "Charlie"]
scores = [85, 92, 78]

# zip pairs up the items from both lists!
for name, score in zip(names, scores):
    print(f"{name} scored {score}")
```
**Output:**
```text
Alice scored 85
Bob scored 92
Charlie scored 78
```
