# 1. While Loops In-Depth

Python has two primitive loop commands: `while` loops, and `for` loops. A `while` loop is used when you want to repeat a set of statements **as long as a condition remains true**. 

---

## 1. The `while` Loop
With the `while` loop we can execute a block of code continuously until the condition becomes False.

**Input:**
```python
i = 1

# This loop will run exactly 5 times
while i <= 5:
    print("The value of i is:", i)
    
    # CRITICAL: We must increment i!
    i += 1 
```
**Output:**
```text
The value of i is: 1
The value of i is: 2
The value of i is: 3
The value of i is: 4
The value of i is: 5
```

### The Infinite Loop Danger
**WARNING:** If you forget to increment the `i` variable in the code above, the condition `i <= 5` will *always* be true, because `i` will always be `1`. The loop will run forever, crashing your program! Always remember to increment your variables inside a `while` loop.

---

## 2. The `else` Statement in While Loops
With the `else` statement, we can run a block of code once when the condition no longer is true. This is a unique feature of Python that doesn't exist in many other languages!

**Input:**
```python
i = 1

while i < 4:
    print(i)
    i += 1
else:
    print("i is no longer less than 4! The loop finished successfully.")
```
**Output:**
```text
1
2
3
i is no longer less than 4! The loop finished successfully.
```

---

## 3. Real World Example: Menu Selection
While loops are often used for taking user input continuously until the user types "quit" or "exit".

**Input:**
```python
user_input = ""

# Keep asking until they type 'quit'
while user_input != "quit":
    user_input = input("Enter a command (type 'quit' to exit): ")
    print("You entered:", user_input)

print("Program terminated.")
```
*(User types 'hello', then 'quit')*
**Output:**
```text
Enter a command (type 'quit' to exit): hello
You entered: hello
Enter a command (type 'quit' to exit): quit
You entered: quit
Program terminated.
```
