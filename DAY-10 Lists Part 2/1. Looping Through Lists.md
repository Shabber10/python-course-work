## 1. Looping Through Lists
You can loop through the list items by using a `for` loop, by index using `range()`, or a `while` loop.

**Input:**
```python
thislist = ["apple", "banana", "cherry"]

# Loop through items
for x in thislist:
    print(x)

# Loop through index numbers
for i in range(len(thislist)):
    print(i, thislist[i])
```
**Output:**
```text
apple
banana
cherry
0 apple
1 banana
2 cherry
```
