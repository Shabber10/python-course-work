# 2. Strings Operations (Length, Looping, Checking)

Because strings act like arrays in Python, there are several powerful built-in operations you can perform on them to analyze their contents.

---

## 1. String Length

To get the total number of characters in a string (including spaces and punctuation), use the `len()` function.

**Input:**
```python
greeting = "Hello, World!"
print("The length of the string is:", len(greeting))

empty_string = ""
print("The length of an empty string is:", len(empty_string))
```
**Output:**
```text
The length of the string is: 13
The length of an empty string is: 0
```

---

## 2. Looping Through a String

Since strings are arrays, you can loop through the characters in a string using a `for` loop. This allows you to inspect or modify every single letter one by one.

**Input:**
```python
# Looping through the word "banana"
print("Spelling out the word:")
for letter in "banana":
    print(letter)
```
**Output:**
```text
Spelling out the word:
b
a
n
a
n
a
```

### Advanced: Counting specific letters
You can use a loop to count how many times a specific letter appears.
**Input:**
```python
word = "programming"
count = 0

for char in word:
    if char == "m":
        count += 1

print("The letter 'm' appears", count, "times.")
```
**Output:**
```text
The letter 'm' appears 2 times.
```

---

## 3. Checking Strings (`in` and `not in`)

To check if a certain phrase or character is present in a string, we can use the `in` keyword. This returns a Boolean (`True` or `False`).

**Input:**
```python
txt = "The best things in life are free!"

print("Is 'free' in the text?", "free" in txt)
print("Is 'expensive' in the text?", "expensive" in txt)
```
**Output:**
```text
Is 'free' in the text? True
Is 'expensive' in the text? False
```

### Using `in` with an `if` statement
This is heavily used in conditional logic to search for keywords in a user's input.
**Input:**
```python
txt = "The best things in life are free!"

if "free" in txt:
    print("Yes, 'free' is present.")
```
**Output:**
```text
Yes, 'free' is present.
```

### Checking if NOT present (`not in`)
To check if a certain phrase or character is NOT present in a string, we can use the `not in` keyword.

**Input:**
```python
txt = "The best things in life are free!"

print("Is 'expensive' NOT in the text?", "expensive" not in txt)

if "expensive" not in txt:
    print("No, 'expensive' is NOT present.")
```
**Output:**
```text
Is 'expensive' NOT in the text? True
No, 'expensive' is NOT present.
```
