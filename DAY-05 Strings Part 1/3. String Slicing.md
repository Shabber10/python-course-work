# 3. String Slicing

You can return a range of characters by using the slicing syntax. Specify the start index and the end index, separated by a colon `:`, to return a part of the string.

This is an extremely powerful feature in Python for parsing data and manipulating text!

---

## 1. Basic Slicing
To slice a string, use the syntax `[start:end]`. 
**Crucial Note:** The `start` index is included, but the `end` index is **NOT included**.

**Input:**
```python
b = "Hello, World!"

# Get the characters from index 2 to index 5 (not included)
print(b[2:5])
```
**Output:**
```text
llo
```
*(Notice how it printed indices 2, 3, and 4, but stopped before 5!)*

---

## 2. Slicing From the Start
If you leave out the start index, the range will automatically start at the first character (index 0).

**Input:**
```python
b = "Hello, World!"

# Get the characters from the start to index 5 (not included)
print(b[:5])
```
**Output:**
```text
Hello
```

---

## 3. Slicing To the End
If you leave out the `end` index, the range will automatically go to the very end of the string.

**Input:**
```python
b = "Hello, World!"

# Get the characters from index 2 all the way to the end
print(b[2:])
```
**Output:**
```text
llo, World!
```

---

## 4. Negative Indexing
Use negative indexes to start the slice from the end of the string.
- `-1` is the last character.
- `-2` is the second to last character, and so on.

**Input:**
```python
b = "Hello, World!"

# From index -5 (the 'o' in World) to index -2 (the 'l' in World)
print(b[-5:-2])
```
**Output:**
```text
orl
```

---

## 5. Advanced: Step Slicing
The full syntax for slicing is actually `[start:end:step]`. The `step` determines how many characters to jump forward. By default, the step is `1`.

**Input:**
```python
alphabet = "abcdefghijklmnopqrstuvwxyz"

# Get every second letter from the start to the end
print(alphabet[::2])

# Trick: Reverse a string using a negative step!
print(alphabet[::-1])
```
**Output:**
```text
acegikmoqsuwy
zyxwvutsrqponmlkjihgfedcba
```
