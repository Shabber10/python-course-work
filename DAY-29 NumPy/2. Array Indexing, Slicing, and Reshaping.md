# 2. Array Indexing, Slicing, and Reshaping

Just like Python lists, you can access and modify elements in a NumPy array. However, NumPy offers significantly more powerful ways to slice multidimensional arrays, filter data conditionally, and change the array's shape.

---

## 1. Indexing and Slicing

### A. One-Dimensional Arrays
Indexing and slicing for 1D arrays work identically to standard Python lists:
`array[start:stop:step]`

**Input:**
```python
import numpy as np

arr = np.array([10, 20, 30, 40, 50])

print("First element :", arr[0])
print("Last element  :", arr[-1])
print("Slice (1 to 4):", arr[1:4])
print("Reverse array :", arr[::-1])
```
**Output:**
```text
First element : 10
Last element  : 50
Slice (1 to 4): [20 30 40]
Reverse array : [50 40 30 20 10]
```

### B. Two-Dimensional Arrays (Matrices)
In standard Python lists of lists, you access a value using `list_of_lists[row][col]`. 
In NumPy, you access values using a single set of brackets with comma-separated coordinates: **`array[row, col]`**.

```text
    2D Array Slicing Coordinate System:
          Col 0   Col 1   Col 2
  Row 0 ┌───────┬───────┬───────┐
        │ [0,0] │ [0,1] │ [0,2] │
        ├───────┼───────┼───────┤
  Row 1 │ [1,0] │ [1,1] │ [1,2] │
        └───────┴───────┴───────┘
```

**Input:**
```python
import numpy as np

matrix = np.array([
    [10, 11, 12],
    [20, 21, 22],
    [30, 31, 32]
])

# Accessing a single element
print("Element at Row 1, Col 2:", matrix[1, 2])

# Slicing: Getting sub-matrices
# matrix[row_start:row_stop, col_start:col_stop]
print("\nFirst 2 rows, last 2 columns:\n", matrix[0:2, 1:3])

# Getting entire rows/columns
print("\nEntire Row 0   :", matrix[0, :])
print("Entire Column 1:", matrix[:, 1])
```
**Output:**
```text
Element at Row 1, Col 2: 22

First 2 rows, last 2 columns:
 [[11 12]
  [21 22]]

Entire Row 0   : [10 11 12]
Entire Column 1: [11 21 31]
```

## 2. Memory Optimization: Copy vs. View

This is one of the most critical concepts in NumPy. 

* **View:** When you slice an array, NumPy does **not** create a new array. Instead, it returns a **view** pointing to the same memory block. Modifying a slice/view will modify the original array!
* **Copy:** To create a completely independent copy of an array, use the **`.copy()`** method.

**Input:**
```python
import numpy as np

original = np.array([1, 2, 3, 4, 5])

# 1. Creating a VIEW via slicing
my_view = original[1:4]  # [2, 3, 4]
my_view[0] = 99         # Modify the view

print("--- View Modification ---")
print("View    :", my_view)
print("Original:", original)  # The original is modified!

# 2. Creating an explicit COPY
original = np.array([1, 2, 3, 4, 5])
my_copy = original[1:4].copy()
my_copy[0] = 99

print("\n--- Copy Modification ---")
print("Copy    :", my_copy)
print("Original:", original)  # The original remains untouched!
```
**Output:**
```text
--- View Modification ---
View    : [99  3  4]
Original: [ 1 99  3  4  5]

--- Copy Modification ---
Copy    : [99  3  4]
Original: [1 2 3 4 5]
```

## 3. Boolean Indexing (Filtering)

Boolean indexing allows you to select elements from an array that satisfy a specific condition.

**Input:**
```python
import numpy as np

ages = np.array([15, 25, 12, 30, 45, 18])

# 1. Apply a logical condition (creates a Boolean Mask)
mask = ages >= 18
print("Boolean Mask :", mask)

# 2. Pass the mask to index the array
adults = ages[mask]
print("Filtered Ages:", adults)

# 3. Direct filtering in one line
print("Ages < 18     :", ages[ages < 18])
```
**Output:**
```text
Boolean Mask : [False  True False  True  True  True]
Filtered Ages: [25 30 45 18]
Ages < 18     : [15 12]
```

## 4. Reshaping and Flattening Arrays

### A. Reshaping (`.reshape()`)
You can change the layout (dimensions) of an array without changing its data using the `.reshape()` method, provided the total number of elements matches.

**Input:**
```python
import numpy as np

# 1D array of 6 elements
arr = np.arange(6)  # [0, 1, 2, 3, 4, 5]

# Reshaping to 2 rows and 3 columns
matrix_2x3 = arr.reshape((2, 3))
print("Reshaped to 2x3:\n", matrix_2x3)

# Short-cut: Using -1 lets NumPy calculate the dimension size automatically
matrix_3x2 = arr.reshape((3, -1))
print("\nReshaped to 3x2 (automatic col layout):\n", matrix_3x2)
```
**Output:**
```text
Reshaped to 2x3:
 [[0 1 2]
 [3 4 5]]

Reshaped to 3x2 (automatic col layout):
 [[0 1]
  [2 3]
  [4 5]]
```

### B. Flattening: `.flatten()` vs. `.ravel()`
Both methods convert a multi-dimensional array into a 1D array, but they differ in memory handling:
- **`flatten()`**: Returns a **copy** (safe but consumes extra memory).
- **`ravel()`**: Returns a **view** where possible (extremely fast and memory efficient).

**Input:**
```python
import numpy as np

grid = np.array([[1, 2], [3, 4]])

flat = grid.flatten()
flat[0] = 99
print("Grid after modifying flatten():\n", grid)  # Unaffected

rav = grid.ravel()
rav[0] = 99
print("\nGrid after modifying ravel()  :\n", grid)  # Original grid modified!
```
**Output:**
```text
Grid after modifying flatten():
 [[1 2]
 [3 4]]

Grid after modifying ravel()  :
 [[99  2]
 [ 3  4]]
```
