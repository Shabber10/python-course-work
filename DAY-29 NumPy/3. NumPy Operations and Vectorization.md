# 3. NumPy Operations and Vectorization

One of the primary advantages of NumPy is its ability to perform fast mathematical operations on entire arrays without writing slow Python `for` loops. This technique is called **Vectorization**.

---

## 1. Element-Wise Operations & Vectorization

When you perform mathematical operations on a NumPy array, the operation is applied to every element automatically.

**Input:**
```python
import numpy as np

arr1 = np.array([1, 2, 3, 4])
arr2 = np.array([10, 20, 30, 40])

# 1. Array-to-Array operations (must be the same shape)
print("Addition      :", arr1 + arr2)
print("Multiplication:", arr1 * arr2)

# 2. Array-to-Scalar operations (Broadcasting a scalar)
print("Add Scalar (5):", arr1 + 5)
print("Square values :", arr1 ** 2)
```
**Output:**
```text
Addition      : [11 22 33 44]
Multiplication: [ 10  40  90 160]
Add Scalar (5): [6 7 8 9]
Square values : [ 1  4  9 16]
```

---

## 2. Understanding Broadcasting

**Broadcasting** is a powerful mechanism that allows NumPy to perform arithmetic operations on arrays of different shapes. The smaller array is effectively "broadcast" (stretched) across the larger array so that they have compatible shapes.

```text
  BROADCASTING IN ACTION (Matrix + Row Vector):
  
    Original Matrix (2x3)          Row Vector (1x3)             Result (2x3)
     ┌───┬───┬───┐                  ┌───┬───┬───┐               ┌───┬───┬───┐
     │ 1 │ 2 │ 3 │                  │10 │20 │30 │               │11 │22 │33 │
     ├───┼───┼───┤         +        └───┴───┴───┘       =       ├───┼───┼───┤
     │ 4 │ 5 │ 6 │             [Automatically stretched         │14 │25 │36 │
     └───┴───┴───┘              to match shape (2x3)]           └───┴───┴───┘
```

### Broadcasting Rules:
Two dimensions are compatible when:
1. They are equal, OR
2. One of them is `1`.

If these rules are not met, NumPy raises a `ValueError: operands could not be broadcast together`.

**Input:**
```python
import numpy as np

# 2D Array of shape (2, 3)
matrix = np.array([
    [1, 2, 3],
    [4, 5, 6]
])

# 1D Vector of shape (3,) - compatible because columns match!
vector = np.array([10, 20, 30])

result = matrix + vector
print("Broadcast Result:\n", result)
```
**Output:**
```text
Broadcast Result:
 [[11 22 33]
 [14 25 36]]
```

## 3. Universal Functions (`ufuncs`)

NumPy provides fast, element-wise functions (called `ufuncs`) that perform mathematical, trigonometric, and logical operations directly on arrays. These are compiled in C, making them extremely fast.

**Input:**
```python
import numpy as np

arr = np.array([1, 4, 9, 16])

print("Square root  :", np.sqrt(arr))
print("Exponential  :", np.exp(arr[:2]))  # e^1 and e^4
print("Trig (Sine)  :", np.sin(arr))
print("Natural Log  :", np.log(arr))
```
**Output:**
```text
Square root  : [1. 2. 3. 4.]
Exponential  : [ 2.71828183 54.59815003]
Trig (Sine)  : [ 0.84147098 -0.7568025  0.41211849 -0.28790332]
Natural Log  : [0.         1.38629436 2.19722458 2.77258872]
```

## 4. Statistical Methods & The Axis Parameter

NumPy arrays have built-in methods to perform descriptive statistical calculations over the entire array or along a specific dimension (axis).

- **`axis=0`**: Computes column-wise (performs operations down the rows).
- **`axis=1`**: Computes row-wise (performs operations across the columns).

```text
               axis=0 (Columns)
                     │
                     ▼
  axis=1 (Rows) ──> [ [ 1,  2,  3 ],
                      [ 4,  5,  6 ] ]
```

**Input:**
```python
import numpy as np

matrix = np.array([
    [10, 20, 30],
    [40, 50, 60]
])

# 1. Aggregate of entire array
print("Sum of all elements :", matrix.sum())
print("Mean of all elements:", matrix.mean())

# 2. Column-wise aggregates (axis=0)
print("\nSum of columns (axis=0) :", matrix.sum(axis=0))
print("Mean of columns (axis=0):", matrix.mean(axis=0))

# 3. Row-wise aggregates (axis=1)
print("\nSum of rows (axis=1)    :", matrix.sum(axis=1))
print("Max value in rows (axis=1):", matrix.max(axis=1))
```
**Output:**
```text
Sum of all elements : 210
Mean of all elements: 35.0

Sum of columns (axis=0) : [50 70 90]
Mean of columns (axis=0): [25. 35. 45.]

Sum of rows (axis=1)    : [ 60 150]
Max value in rows (axis=1): [30 60]
```
