# 1. Introduction to NumPy and NDArrays

To perform high-performance numerical computations in Python, standard lists are not efficient enough. For scientific computing, data analysis, and machine learning, we use **NumPy** (Numerical Python)вҖ”the foundational package for scientific computing in Python.

---

## 1. What is NumPy and Why Use It?

NumPy provides a powerful N-dimensional array object called **`ndarray`** (N-dimensional array), along with tools for working with these arrays. 

### Why not just use Python Lists?
While Python lists are extremely flexible because they can store different data types, this flexibility comes at a massive cost in terms of performance and memory.

```text

  в”Ңв”Җв”Җв”Җв”¬в”Җв”Җв”Җв”¬в”Җв”Җв”Җв”¬в”Җв”Җв”Җв”җ
```

| Feature | Python Lists | NumPy Arrays (`ndarray`) |
|:---|:---|:---|
| **Memory Allocation** | Scattered in memory (pointers to objects). | Contiguous block of memory. |
| **Data Types** | Can hold mixed types (heterogeneous). | Must hold the same type (homogeneous). |
| **Operations** | Element-wise operations require loops. | Vectorized operations (no loops, runs in C). |
| **Performance** | Slower for large datasets. | Extremely fast (up to 100x faster). |

## 3. Creating NumPy Arrays

There are several ways to initialize a NumPy array depending on your needs.

### A. Creating from Python Lists
You can convert standard Python lists or tuples into NumPy arrays using `np.array()`.

**Input:**
```python
import numpy as np

# 1D Array
list_1d = [1, 2, 3, 4, 5]
arr_1d = np.array(list_1d)
print("1D Array:", arr_1d)
print("Type    :", type(arr_1d))

# 2D Array (Matrix)
list_2d = [[1, 2, 3], [4, 5, 6]]
arr_2d = np.array(list_2d)
print("\n2D Array (Matrix):\n", arr_2d)
```
**Output:**
```text
1D Array: [1 2 3 4 5]
Type    : <class 'numpy.ndarray'>

2D Array (Matrix):
 [[1 2 3]
 [4 5 6]]
```

### B. Built-in Array Initializers
NumPy provides functions to generate arrays filled with specific values, sequences, or patterns.

**Input:**
```python
import numpy as np

# 1. Zeros: Create an array filled with 0.0 (float by default)
zeros_arr = np.zeros((2, 3))
print("Zeros Array:\n", zeros_arr)

# 2. Ones: Create an array filled with 1.0
ones_arr = np.ones((3, 2), dtype=int)  # Forcing integer type
print("\nOnes Array (Integer):\n", ones_arr)

# 3. Arange: Like range(), but returns an array
range_arr = np.arange(0, 10, 2)  # (start, stop, step)
print("\nArange Array:", range_arr)

# 4. Linspace: Creates N evenly spaced values over a specified interval
linspace_arr = np.linspace(0, 1, 5)  # (start, stop, number_of_samples)
print("\nLinspace Array:", linspace_arr)
```
**Output:**
```text
Zeros Array:
 [[0. 0. 0.]
 [0. 0. 0.]]

Ones Array (Integer):
 [[1 1]
  [1 1]
  [1 1]]

Arange Array: [0 2 4 6 8]

Linspace Array: [0.   0.25 0.5  0.75 1.  ]
```

## 4. Crucial Array Attributes

Every `ndarray` has several attributes that describe its size, shape, and data types:

- **`.ndim`**: The number of dimensions (axes) of the array.
- **`.shape`**: A tuple of integers indicating the size of the array in each dimension.
- **`.size`**: The total number of elements in the array.
- **`.dtype`**: The data type of the elements in the array (e.g., `int32`, `float64`).
- **`.itemsize`**: The size in bytes of a single array element.

**Input:**
```python
import numpy as np

arr = np.array([[10, 20, 30], [40, 50, 60]])

print("Array structure:\n", arr)
print("Dimensions (ndim)   :", arr.ndim)
print("Shape (shape)       :", arr.shape)
print("Total elements (size):", arr.size)
print("Data Type (dtype)   :", arr.dtype)
print("Item size (bytes)   :", arr.itemsize)
```
**Output:**
```text
Array structure:
 [[10 20 30]
 [40 50 60]]
Dimensions (ndim)   : 2
Shape (shape)       : (2, 3)
Total elements (size): 6
Data Type (dtype)   : int32
Item size (bytes)   : 4
```

> [!NOTE]
> **Homogeneity Rule:**
> If you try to mix data types (e.g., integers and strings) in `np.array()`, NumPy will automatically upcast/coerce all elements to a single common type (usually strings or floats) to maintain homogeneity.
> ```python
> arr = np.array([1, 2.5, "three"])
> print(arr.dtype)  # Output: <U32 (unicode string)
> ```
