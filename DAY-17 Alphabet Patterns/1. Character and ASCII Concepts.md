# 1. Character and ASCII Concepts

Before printing complex alphabet designs, we must understand how Python represents and manipulates individual characters under the hood.

---

## 1. ASCII Codes, `ord()` and `chr()`
In computer systems, every character is represented by a unique integer. The most common standard is **ASCII (American Standard Code for Information Interchange)**.

In Python, we have two built-in functions to convert between characters and their integer values:
1. `ord(char)`: Returns the integer value (ASCII) of a given character.
2. `chr(int)`: Returns the character string of a given integer value (ASCII).

### Useful ASCII Values to Remember:
- **`'A'` to `'Z'`**: `65` to `90`
- **`'a'` to `'z'`**: `97` to `122`

**Input:**
```python
# ord() converts character -> integer
print("ASCII value of 'A' is:", ord('A'))
print("ASCII value of 'B' is:", ord('B'))
print("ASCII value of 'a' is:", ord('a'))

# chr() converts integer -> character
print("Character for ASCII 65 is:", chr(65))
print("Character for ASCII 90 is:", chr(90))
print("Character for ASCII 97 is:", chr(97))
```
**Output:**
```text
ASCII value of 'A' is: 65
ASCII value of 'B' is: 66
ASCII value of 'a' is: 97
Character for ASCII 65 is: A
Character for ASCII 90 is: Z
Character for ASCII 97 is: a
```

## 2. Letter-based Pattern Examples
Using `chr()` and loops, we can print alphabet patterns similarly to number patterns.

### A. Fixed Letters per Row
Each row prints the same character. Row `i` (0 to 4) uses ASCII `65 + i` (representing `'A'`, `'B'`, `'C'`, etc.).

**Input:**
```python
n = 5
for i in range(n):
    letter = chr(65 + i)
    for j in range(i + 1):
        print(letter, end=" ")
    print()
```
**Output:**
```text
A 
B B 
C C C 
D D D D 
E E E E E 
```

### B. Increasing Letters per Row
Each row starts at `'A'` (ASCII `65`) and increments the character by `1` per column.

**Input:**
```python
n = 5
for i in range(n):
    for j in range(i + 1):
        letter = chr(65 + j)
        print(letter, end=" ")
    print()
```
**Output:**
```text
A 
A B 
A B C 
A B C D 
A B C D E 
```

### C. Continuous Alphabet Triangle
Like Floyd's Triangle, we start a counter at ASCII `65` and increment it for each character printed.

**Input:**
```python
n = 4
ascii_val = 65
for i in range(n):
    for j in range(i + 1):
        print(chr(ascii_val), end=" ")
        ascii_val += 1
    print()
```
**Output:**
```text
A 
B C 
D E F 
G H I J 
```
