# 2. Printing A-Z Letter Shapes

Printing letter shapes (A-Z) using stars (`*`) is a classic exercise in nested loops and complex logical conditions. It is done by visualizing each letter on a grid (typically 7 rows by 5 columns).

---

## 1. Grid Logic (7x5)
We treat each letter as a 2D coordinates grid:
- **Rows (`i`)** from `0` to `6` (vertical axis).
- **Columns (`j`)** from `0` to `4` (horizontal axis).

### 📍 Visualizing Letter 'A' Coordinates
```text
Row i (0-6)
    0  ->   * * *     (Row 0, Cols 1,2,3)
    1  ->  *     *    (Row 1, Cols 0 & 4)
    2  ->  *     *    (Row 2, Cols 0 & 4)
    3  ->  *****    (Row 3, Cols 0,1,2,3,4 - Middle Bar)
    4  ->  *     *    (Row 4, Cols 0 & 4)
    5  ->  *     *    (Row 5, Cols 0 & 4)
    6  ->  *     *    (Row 6, Cols 0 & 4)
           |     |
Col j ->   0     4
```

By checking the values of `i` and `j` inside our loops, we decide whether to print a star `*` or a space `" "`.

---

## 2. Examples of Common Letters

### Letter 'A'
- **Vertical Bars:** Columns `0` and `4` (excluding the top row, so `i != 0`).
- **Top Row:** Row `0` (excluding the corners, so `0 < j < 4`).
- **Middle Bar:** Row `3` (all columns).

**Input:**
```python
for i in range(7):
    for j in range(5):
        if ((j == 0 or j == 4) and i != 0) or (i == 0 and 0 < j < 4) or (i == 3):
            print("*", end="")
        else:
            print(" ", end="")
    print()
```
**Output:**
```text
 *** 
*   *
*   *
*****
*   *
*   *
*   *
```

---

### Letter 'B'
- **Left Bar:** Column `0` (all rows).
- **Horizontal Bars:** Rows `0`, `3`, and `6` (columns `0` to `3`).
- **Right Curves:** Column `4` (rows `1`, `2`, `4`, `5`).

**Input:**
```python
for i in range(7):
    for j in range(5):
        if (j == 0) or (j == 4 and i in [1, 2, 4, 5]) or (i in [0, 3, 6] and 0 <= j < 4):
            print("*", end="")
        else:
            print(" ", end="")
    print()
```
**Output:**
```text
**** 
*   *
*   *
**** 
*   *
*   *
**** 
```

---

### Letter 'C'
- **Left Curve:** Column `0` (excluding the top and bottom corners: `0 < i < 6`).
- **Top & Bottom Rows:** Rows `0` and `6` (excluding the first column: `j > 0`).

**Input:**
```python
for i in range(7):
    for j in range(5):
        if (j == 0 and 0 < i < 6) or ((i == 0 or i == 6) and j > 0):
            print("*", end="")
        else:
            print(" ", end="")
    print()
```
**Output:**
```text
 ****
*    
*    
*    
*    
*    
 ****
```

## 3. Comprehensive Script for A-Z
Here is a single, complete Python script containing logic functions to print any letter from A to Z on a 7x5 grid.

**Input:**
```python
def print_letter(char):
    char = char.upper()
    for i in range(7):
        for j in range(5):
            is_star = False
            if char == 'A':
                is_star = ((j == 0 or j == 4) and i != 0) or (i == 0 and 0 < j < 4) or (i == 3)
            elif char == 'B':
                is_star = (j == 0) or (j == 4 and i in [1, 2, 4, 5]) or (i in [0, 3, 6] and j < 4)
            elif char == 'C':
                is_star = (j == 0 and 0 < i < 6) or ((i == 0 or i == 6) and j > 0)
            elif char == 'D':
                is_star = (j == 0) or (j == 4 and i in [1, 2, 3, 4, 5]) or (i in [0, 6] and j < 4)
            elif char == 'E':
                is_star = (j == 0) or (i in [0, 3, 6])
            elif char == 'F':
                is_star = (j == 0) or (i in [0, 3])
            elif char == 'G':
                is_star = (j == 0 and 0 < i < 6) or ((i == 0 or i == 6) and j > 0) or (i == 3 and j >= 2) or (j == 4 and i >= 3)
            elif char == 'H':
                is_star = (j == 0 or j == 4) or (i == 3)
            elif char == 'I':
                is_star = (i == 0 or i == 6) or (j == 2)
            elif char == 'J':
                is_star = (i == 0) or (j == 2 and i < 6) or (i == 6 and j < 2) or (i == 5 and j == 0)
            elif char == 'K':
                is_star = (j == 0) or (j == 3 - i and i <= 3) or (j == i - 3 and i >= 3)
            elif char == 'L':
                is_star = (j == 0) or (i == 6)
            elif char == 'M':
                is_star = (j == 0 or j == 4) or (i == 1 and j in [1, 3]) or (i == 2 and j == 2)
            elif char == 'N':
                is_star = (j == 0 or j == 4) or (i == j)
            elif char == 'O':
                is_star = ((j == 0 or j == 4) and 0 < i < 6) or ((i == 0 or i == 6) and 0 < j < 4)
            elif char == 'P':
                is_star = (j == 0) or (j == 4 and i in [1, 2]) or (i in [0, 3] and j < 4)
            elif char == 'Q':
                is_star = ((j == 0 or j == 4) and 0 < i < 5) or ((i == 0 or i == 5) and 0 < j < 4) or (i == j and i >= 4)
            elif char == 'R':
                is_star = (j == 0) or (j == 4 and i in [1, 2]) or (i in [0, 3] and j < 4) or (i >= 4 and j == i - 2)
            elif char == 'S':
                is_star = (i in [0, 3, 6]) or (j == 0 and 0 < i < 3) or (j == 4 and 3 < i < 6)
            elif char == 'T':
                is_star = (i == 0) or (j == 2)
            elif char == 'U':
                is_star = ((j == 0 or j == 4) and i < 6) or (i == 6 and 0 < j < 4)
            elif char == 'V':
                is_star = (j == 0 and i < 5) or (j == 4 and i < 5) or (i == 5 and (j == 1 or j == 3)) or (i == 6 and j == 2)
            elif char == 'W':
                is_star = (j == 0 or j == 4) or (i == 5 and j in [1, 3]) or (i == 4 and j == 2)
            elif char == 'X':
                is_star = (i == j) or (j == 4 - i)
            elif char == 'Y':
                is_star = (i <= 3 and (j == i or j == 4 - i)) or (i > 3 and j == 2)
            elif char == 'Z':
                is_star = (i == 0 or i == 6) or (j == 4 - i)
            
            print("*" if is_star else " ", end="")
        print()
    print()

# Test printing A, B, and C
print_letter('A')
print_letter('B')
print_letter('C')
```
