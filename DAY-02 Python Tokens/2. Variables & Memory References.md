# 2. Variables & Memory References

Variables are containers for storing data values. Unlike other programming languages, Python has no command for declaring a variable. A variable is created the moment you first assign a value to it!

---

## 1. Creating Variables
To create a variable, you simply write the variable name, followed by an equals sign `=`, followed by the value you want to store.

**Input:**
```python
x = 5         # Python automatically knows this is an integer
y = "John"    # Python automatically knows this is a string
print(x)
print(y)
```
**Output:**
```text
5
John
```

---

## 2. Dynamic Typing & Type Reassignment
Variables in Python are **dynamically typed**. This means you do not need to declare their type, and they can even change type after they have been set!

**Input:**
```python
x = 4       # x is of type int
print("Initial x:", x)

x = "Sally" # x is now of type str! Python handles this perfectly.
print("Reassigned x:", x)
```
**Output:**
```text
Initial x: 4
Reassigned x: Sally
```

---

## 3. Multiple Variables in One Line
Python allows you to assign values to multiple variables in a single line. This is a very "Pythonic" way to write clean, concise code.

**Input:**
```python
# Assigning different values to different variables
x, y, z = "Orange", "Banana", "Cherry"
print(x, y, z)

# Assigning the SAME value to multiple variables
a = b = c = "Apple"
print(a, b, c)
```
**Output:**
```text
Orange Banana Cherry
Apple Apple Apple
```

---

## 4. Memory References (The `id()` Function)
Under the hood, Python variables do not actually store data. They store a **memory reference** (a pointer) to an object in the computer's RAM.

You can actually see the exact memory address of where a value is stored using Python's built-in `id()` function.

**Input:**
```python
a = 100
b = 100

print("Value of a:", a)
print("Memory address of a:", id(a))

print("Value of b:", b)
print("Memory address of b:", id(b))

# Because 100 is the same value, Python optimizes memory by pointing BOTH 'a' and 'b' to the exact same location!
```
**Output:**
```text
Value of a: 100
Memory address of a: 140723849315816
Value of b: 100
Memory address of b: 140723849315816
```

---

## 5. Deleting Variables (`del`)
If you want to completely erase a variable from memory, you can use the `del` keyword.

**Input:**
```python
temp_var = "Temporary Data"
del temp_var
# print(temp_var) # If we try to print it now, Python throws a NameError!
```
**Output:**
```text
# Nothing prints because the variable was successfully destroyed.
```
