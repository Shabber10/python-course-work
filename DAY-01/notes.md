# Day 1: Introduction to Python

## What is Python?
Python is a high-level, interpreted programming language known for its readability, simplicity, and broad range of applications (web development, data science, automation, etc.).

## Printing to the Console
The `print()` function is used to output text or data to the console.
```python
print("Hello, World!")
print(10 + 20)
```

## Variables
Variables are containers for storing data values. You don't need to declare their type explicitly.
```python
name = "Alice"
age = 25
print("My name is", name, "and I am", age, "years old.")
```

## Taking User Input
The `input()` function pauses program execution to allow the user to type something in.
```python
# The input is always read as a string
user_name = input("Enter your name: ")
print("Welcome to Python,", user_name)
```
