# 2. Advanced Patterns (Pyramids and Diamonds)

Advanced patterns involve multiple spaces and varying math logic for columns. To print these successfully, we break each row into two parts:
1. **Leading Spaces**
2. **Stars / Symbols**

---

## 1. Full Pyramid Pattern
For a pyramid of size `n`:
- Row `i` (0-indexed) requires `n - i - 1` spaces.
- Row `i` requires `2 * i + 1` stars.

**Input:**
```python
n = 5
for i in range(n):
    # 1. Print leading spaces
    for j in range(n - i - 1):
        print(" ", end="")
    # 2. Print stars
    for k in range(2 * i + 1):
        print("*", end="")
    print()
```
**Output:**
```text
    *
   ***
  *****
 *******
*********
```

### 📊 Pyramid Math Logic Table (for `n = 5`)
Let's see how the formula works mathematically:

| Row Index `i` | Spaces formula: `n - i - 1` | Spaces Printed | Stars formula: `2 * i + 1` | Stars Printed |
| :---: | :---: | :---: | :---: | :---: |
| **`0`** | `5 - 0 - 1` | `4` | `2 * 0 + 1` | `1` |
| **`1`** | `5 - 1 - 1` | `3` | `2 * 1 + 1` | `3` |
| **`2`** | `5 - 2 - 1` | `2` | `2 * 2 + 1` | `5` |
| **`3`** | `5 - 3 - 1` | `1` | `2 * 3 + 1` | `7` |
| **`4`** | `5 - 4 - 1` | `0` | `2 * 4 + 1` | `9` |

---

## 2. Inverted Pyramid Pattern
For an inverted pyramid of size `n`:
- Row `i` (0-indexed) requires `i` spaces.
- Row `i` requires `2 * (n - i) - 1` stars.

**Input:**
```python
n = 5
for i in range(n):
    # 1. Print leading spaces
    for j in range(i):
        print(" ", end="")
    # 2. Print stars
    for k in range(2 * (n - i) - 1):
        print("*", end="")
    print()
```
**Output:**
```text
*********
 *******
  *****
   ***
    *
```

---

## 3. Diamond Pattern
A diamond pattern is created by combining the logic of a **Full Pyramid** (upper half) and an **Inverted Pyramid** (lower half).

**Input:**
```python
n = 5  # Represents half-height of the diamond

# Upper Half (including middle row)
for i in range(n):
    for j in range(n - i - 1):
        print(" ", end="")
    for k in range(2 * i + 1):
        print("*", end="")
    print()

# Lower Half
for i in range(n - 2, -1, -1):
    for j in range(n - i - 1):
        print(" ", end="")
    for k in range(2 * i + 1):
        print("*", end="")
    print()
```
**Output:**
```text
    *
   ***
  *****
 *******
*********
 *******
  *****
   ***
    *
```

---

## 4. Hollow Diamond Pattern
A hollow diamond only prints stars at the boundaries of the shape and uses spaces inside.

**Input:**
```python
n = 5

# Upper Half
for i in range(n):
    # Outer spaces
    for j in range(n - i - 1):
        print(" ", end="")
    # Stars and inner spaces
    for k in range(2 * i + 1):
        if k == 0 or k == 2 * i:
            print("*", end="")
        else:
            print(" ", end="")
    print()

# Lower Half
for i in range(n - 2, -1, -1):
    # Outer spaces
    for j in range(n - i - 1):
        print(" ", end="")
    # Stars and inner spaces
    for k in range(2 * i + 1):
        if k == 0 or k == 2 * i:
            print("*", end="")
        else:
            print(" ", end="")
    print()
```
**Output:**
```text
    *
   * *
  *   *
 *     *
*       *
 *     *
  *   *
   * *
    *
```
