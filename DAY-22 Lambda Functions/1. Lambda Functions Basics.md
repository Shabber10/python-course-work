# 1. Lambda Functions Basics

A **lambda function** (also called an **anonymous function**) is a small, unnamed function in Python. It is defined using the `lambda` keyword instead of the standard `def` keyword.

---

## 1. Syntax of a Lambda Function
A lambda function can take any number of arguments, but it can only have **one single expression**. The expression is evaluated and automatically returned.

### Syntax:
```python
lambda arguments: expression
```

---

## 2. Comparing `def` vs. `lambda`

### Example A: Addition function
Let's define a function that adds two numbers.

#### Standard function (`def`):
```python
def add(x, y):
    return x + y

print(add(5, 3))
```

#### Lambda function (`lambda`):
**Input:**
```python
# We can assign the lambda function to a variable to call it
add_lambda = lambda x, y: x + y

print(add_lambda(5, 3))
```
**Output:**
```text
8
```

---

## 3. Key Characteristics of Lambdas
1. **Anonymous:** They do not require a name (though you can assign them to variables for reference).
2. **Single Expression:** They cannot contain loops, multiple statements, or complex logic. They are restricted to a single returned expression.
3. **Implicit Return:** The result of the expression is automatically returned; you do not (and cannot) write the `return` keyword inside a lambda.

**Input:**
```python
# Lambda checking if a number is even (returns boolean)
is_even = lambda n: n % 2 == 0

print("Is 4 even?", is_even(4))
print("Is 7 even?", is_even(7))
```
**Output:**
```text
Is 4 even? True
Is 7 even? False
```

---

## 4. Why Use Lambda Functions?
Lambda functions are best used as **temporary, throwaway functions** that are passed as arguments to higher-order functions (functions that take other functions as inputs). 

Using a lambda saves you from having to write a separate `def` function block elsewhere in your file for a trivial, one-time operation.
