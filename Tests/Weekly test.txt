Q16: You are given n different online courses numbered from 1 to n. Each course i is represented by an array courses[i] = [duration_i, lastDay_i], where: duration_i is the number of consecutive days it takes to complete the course. lastDay_i is the last day by which the course must be finished (inclusive). You start on day 1 and cannot take two or more courses simultaneously. Return the maximum number of courses you can take. You need to write a program that reads input from standard input and prints the output to standard output. Input Format : The first line contains an integer n, the number of courses. The next n lines each contain two space-separated integers: duration_i and lastDay_i. Output Format : Print a single integer, the maximum number of courses you can take.
Test Cases:
0 Passed
5 Failed
Marks
0/10
Your Code

a=int(input())
b=list(map(int,input().split()) for _ in range(a))
print("".join(b))
Correct Solution

n = int(input())
courses = []
for _ in range(n):
    duration, lastDay = map(int, input().split())
    courses.append([duration, lastDay])
courses.sort(key=lambda x: x[1])
taken_courses = []
total_time = 0
for duration, lastDay in courses:
    if total_time + duration <= lastDay:
        taken_courses.append(duration)
        total_time += duration
    else:
        if taken_courses and max(taken_courses) > duration:
            longest = max(taken_courses)
            taken_courses.remove(longest)
            total_time -= longest
            taken_courses.append(duration)
            total_time += duration
print(len(taken_courses))

Q22: Suppose an array of length n sorted in ascending order is rotated between 1 and n times. For example, the array [0,1,4,4,5,6,7] might become: [4,5,6,7,0,1,4] if it was rotated 4 times. [0,1,4,4,5,6,7] if it was rotated 7 times. Rotating an array [a[0], a[1], ..., a[n-1]] 1 time results in [a[n-1], a[0], a[1], ..., a[n-2]]. Given the sorted rotated array that may contain duplicates, print the minimum element of this array. Input Format : First line: an integer n — the size of the array. Second line: n space-separated integers — the elements of the array. Output Format : Print a single integer — the minimum element of the array.
Test Cases:
- Passed
- Failed
Marks
0/5
Correct Solution

n = int(input())
nums = list(map(int, input().split()))
left, right = 0, n - 1
while left < right:
    mid = left + (right - left) // 2
    if nums[mid] < nums[right]:
        right = mid
    elif nums[mid] > nums[right]:
        left = mid + 1
    else:
        right -= 1
print(nums[left])

Q17: You are given an integer n representing the number of pairs of parentheses. Your task is to generate all possible combinations of well-formed parentheses. A well-formed parentheses string is one where: Every opening bracket '(' has a matching closing bracket ')' At no point should a closing bracket ')' exceed the number of opening brackets '(' Return the list of valid combinations in any order. Input Format : A single integer n, the number of pairs of parentheses. Output Format : Print all the valid parentheses combinations. Each combination must be printed on a new line. Order does not matter.
Test Cases:
5 Passed
0 Failed
Marks
10/10
Your Code

def generate_parentheses(n):
  def backtrack(s, left, right):
    if len(s)==2*n:
      print(s)
      return
    if left<n:
      backtrack(s+"(",left+1,right)
    if right<left:
      backtrack(s+")",left,right+1)
  backtrack("",0,0)

n=int(input())
generate_parentheses(n)
