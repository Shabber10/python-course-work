# 2. Writing and Appending Files

Writing data to files is how we export reports, save game states, and log system events. Depending on how you want to handle existing data in a file, Python offers several distinct modes.

---

## 1. Understanding Writing Modes

When opening a file for writing, choosing the correct mode dictates whether you preserve, overwrite, or protect your data:

| Mode | Name | Behavior |
| :---: | :--- | :--- |
| **`'w'`** | Write | Overwrites the file if it already exists. Creates a new file if it doesn't exist. |
| **`'a'`** | Append | Adds new data to the **end** of the file. Preserves existing data. Creates the file if it doesn't exist. |
| **`'x'`** | Exclusive Creation | Creates a new file. If the file **already exists**, Python raises a `FileExistsError` to prevent accidental overwrites. |
| **`'r+'`** | Read + Write | Opens an existing file for both reading and writing. The cursor starts at the beginning of the file. |
| **`'w+'`** | Write + Read | Opens a file for reading and writing, but **overwrites** existing content. |
| **`'a+'`** | Append + Read | Opens a file for reading and writing. All write operations append to the end of the file. |

---

## 2. Writing Methods

Python provides two main methods to write text to a file:

### 1. `.write(string)`
Writes a single string to the file. 

> [!IMPORTANT]
> `.write()` does **not** automatically add a newline character (`\n`) to the end of the text. You must explicitly include `\n` in your string if you want a line break.

**Input:**
```python
# Open in write mode ('w') - will create the file
with open("output.txt", "w") as file:
    file.write("Hello World!\n")
    file.write("This is the second line.\n")
    
# Let's verify the file content
with open("output.txt", "r") as file:
    print(file.read())
```
**Output:**
```text
Hello World!
This is the second line.
```

### 2. `.writelines(list_of_strings)`
Writes an iterable (like a list or tuple) of strings to the file. Like `.write()`, it does **not** add newlines between the strings in the list automatically.

**Input:**
```python
lines_to_write = [
    "Row 1 text.\n",
    "Row 2 text.\n",
    "Row 3 text.\n"
]

with open("matrix.txt", "w") as file:
    file.writelines(lines_to_write)

# Verify
with open("matrix.txt", "r") as file:
    print(file.read())
```
**Output:**
```text
Row 1 text.
Row 2 text.
Row 3 text.
```

---

## 3. Appending to a File (`'a'`)

If we run the write mode code `open("output.txt", "w")` again, all our previous sentences will be instantly deleted and replaced. To add data while preserving what is already there, we use append mode (`'a'`).

**Input:**
```python
# Open in append mode ('a')
with open("output.txt", "a") as file:
    file.write("This line was appended later.\n")

# Verify
with open("output.txt", "r") as file:
    print(file.read())
```
**Output:**
```text
Hello World!
This is the second line.
This line was appended later.
```

## 4. Practical Project: Application Event Logger

Let's apply file writing and appending by building a reusable logging utility. This logger will record application events to a file named `app.log` along with a precise timestamp (using the `datetime` module we covered in Day 24).

**Input:**
```python
from datetime import datetime

class AppLogger:
    def __init__(self, log_filename="app.log"):
        self.log_filename = log_filename

    def log_event(self, level, message):
        # 1. Generate current timestamp: [YYYY-MM-DD HH:MM:SS]
        timestamp = datetime.now().strftime("%Y-%m-%d %H:%M:%S")
        
        # 2. Format the log line
        log_line = f"[{timestamp}] [{level.upper()}] {message}\n"
        
        # 3. Append to the log file
        with open(self.log_filename, "a") as file:
            file.write(log_line)

# Create logger instance
logger = AppLogger("test_app.log")

# Log system events
logger.log_event("info", "Application initialized.")
logger.log_event("warning", "Database connection timeout. Retrying...")
logger.log_event("error", "Failed to write user profile. Database connection lost.")

# Read and print the log file
print("--- Application Log File Contents ---")
with open("test_app.log", "r") as file:
    print(file.read())
```
**Output:**
```text
--- Application Log File Contents ---
[2026-07-09 09:30:15] [INFO] Application initialized.
[2026-07-09 09:30:16] [WARNING] Database connection timeout. Retrying...
[2026-07-09 09:30:17] [ERROR] Failed to write user profile. Database connection lost.
```
