# Advanced Methods and Functions

Python provides an enormous suite of methods and built-in functions to process Lists. This document serves as the master reference for absolutely everything you can do to a List.

## 12. Deep Dive: Advanced Methods

### `count()` and `index()`
These methods help you analyze the contents of your list.
**Input:**
```python
fruits = ['apple', 'banana', 'cherry', 'apple', 'apple']

# Count how many times 'apple' appears
print("Apples:", fruits.count('apple'))

# Find the index position of 'cherry'
print("Cherry index:", fruits.index('cherry'))
```

**Output:**
```text
Apples: 3
Cherry index: 2
```


### `copy()`
If you type `list2 = list1`, changing `list1` will ALSO change `list2` because they share the same memory! Use `.copy()` to safely clone a list.
**Input:**
```python
fruits = ["apple", "banana"]
safe_copy = fruits.copy()

fruits.clear()
print("Safe Copy survived!", safe_copy)
```

**Output:**
```text
Safe Copy survived! ['apple', 'banana']
```


### `reverse()`
Permanently reverses the current list.
**Input:**
```python
numbers = [1, 2, 3]
numbers.reverse()
print("Reversed:", numbers)
```

**Output:**
```text
Reversed: [3, 2, 1]
```

## 13. Deep Dive: Advanced Built-in Functions

### `min()`, `max()`, `sum()`
**Input:**
```python
nums = [10, 50, 5, 100]
print("Max:", max(nums))
print("Min:", min(nums))
print("Sum:", sum(nums))
```

**Output:**
```text
Max: 100
Min: 5
Sum: 165
```


### `sorted()` and `reversed()`
Unlike `.sort()` and `.reverse()`, these return BRAND NEW lists instead of modifying the original.
**Input:**
```python
original = [3, 1, 2]
new_sorted = sorted(original)
print("Original is safe:", original)
print("New Sorted:", new_sorted)

# reversed() returns an iterator, so we cast it to a list
new_reversed = list(reversed(original))
print("New Reversed:", new_reversed)
```

**Output:**
```text
Original is safe: [3, 1, 2]
New Sorted: [1, 2, 3]
New Reversed: [2, 1, 3]
```


### `enumerate()`
Automatically generates the index number alongside the value in a `for` loop!
**Input:**
```python
fruits = ["apple", "banana", "cherry"]
for index, value in enumerate(fruits):
    print("Index:", index, "Value:", value)
```

**Output:**
```text
Index: 0 Value: apple
Index: 1 Value: banana
Index: 2 Value: cherry
```


### `zip()`
Combines two lists element-by-element like a zipper on a jacket!
**Input:**
```python
names = ["Alice", "Bob"]
ages = [25, 30]

zipped = list(zip(names, ages))
print("Zipped:", zipped)
```

**Output:**
```text
Zipped: [('Alice', 25), ('Bob', 30)]
```


### `map()` and `filter()`
Apply a function to every item, or filter items based on a function.
**Input:**
```python
def is_even(num):
    return num % 2 == 0

def double_it(num):
    return num * 2

numbers = [1, 2, 3, 4]

print("Mapped (Doubled):", list(map(double_it, numbers)))
print("Filtered (Evens):", list(filter(is_even, numbers)))
```

**Output:**
```text
Mapped (Doubled): [2, 4, 6, 8]
Filtered (Evens): [2, 4]
```

## 14. Joining Lists

There are several ways to join, or concatenate, two or more lists in Python.

### Method 1: Using the `+` Operator
The easiest way is to use the `+` operator, which returns a brand new list.
**Input:**
```python
list1 = ["a", "b", "c"]
list2 = [1, 2, 3]

list3 = list1 + list2
print("Joined List:", list3)
```
**Output:**
```text
Joined List: ['a', 'b', 'c', 1, 2, 3]
```

### Method 2: Using `extend()`
If you want to add the elements from one list *into* a pre-existing list (without creating a new variable), use the `extend()` method.
**Input:**
```python
list1 = ["a", "b", "c"]
list2 = [1, 2, 3]

list1.extend(list2)
print("Extended List1:", list1)
```
**Output:**
```text
Extended List1: ['a', 'b', 'c', 1, 2, 3]
```

## 15. Advanced Input Parsing (The Map Function)

When building terminal applications or participating in competitive programming, you often need to ask the user for a sequence of numbers on a single line (e.g., `10 20 30 40`), and instantly convert them into a List of Integers.

This is done by combining `input()`, `.split()`, `map()`, and `list()`!

### Taking a List of Numbers
**Input:**
```python
# The user types: 10 20 30 40
user_input = "10 20 30 40" # Simulating input()

# 1. split() breaks the string into a list of strings: ["10", "20", "30", "40"]
# 2. map(int, ...) converts each string into an integer.
# 3. list(...) finalizes it into a Python List.
number_list = list(map(int, user_input.split()))

print(number_list)
```
**Output:**
```text
[10, 20, 30, 40]
```

### Joining Numbers Back to a String
If you try to `.join()` a list of integers, Python will crash. You must map them back to strings first!

**Input:**
```python
number_list = [10, 20, 30, 40]

# 1. map(str, ...) converts each integer into a string.
# 2. "".join(...) concatenates them all together.
final_string = "-".join(map(str, number_list))

print(final_string)
```
**Output:**
```text
10-20-30-40
```
