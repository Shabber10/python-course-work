# 3. Advanced Methods and Functions

Python provides an enormous suite of methods and built-in functions to process Lists. This document serves as the master reference for absolutely everything you can do to a List.

---

## 1. Master Reference Table
Here are all 11 List Methods and 12 Built-in Functions that operate on Lists.

### List Methods
| Method | Description |
|---|---|
| `append()` | Adds an element at the end of the list. (Covered in Part 2) |
| `clear()` | Removes all the elements from the list. (Covered in Part 2) |
| `copy()` | Returns a shallow copy of the list. |
| `count()` | Returns the number of elements with the specified value. |
| `extend()` | Add the elements of a list (or any iterable) to the end. (Covered in Part 2) |
| `index()` | Returns the index of the first element with the specified value. |
| `insert()` | Adds an element at the specified position. (Covered in Part 2) |
| `pop()` | Removes the element at the specified position. (Covered in Part 2) |
| `remove()` | Removes the first item with the specified value. (Covered in Part 2) |
| `reverse()` | Reverses the order of the list permanently. |
| `sort()` | Sorts the list permanently. (Covered in Part 2) |

### Built-in Functions for Lists
| Function | Description |
|---|---|
| `len()` | Returns the length of the list. (Covered in Part 1) |
| `any()` | Returns True if any item is True. (Covered in Part 1) |
| `all()` | Returns True if all items are True. (Covered in Part 1) |
| `min()` | Returns the lowest value. |
| `max()` | Returns the highest value. |
| `sum()` | Returns the sum of all numerical items. |
| `sorted()` | Returns a brand new sorted list (does not change original). |

| `map()` | Applies a function to all items in the list. |

## 2. Deep Dive: Advanced Methods

### `count()` and `index()`
These methods help you analyze the contents of your list.
**Input:**
```python
fruits = ['apple', 'banana', 'cherry', 'apple', 'apple']

# Count how many times 'apple' appears
print("Apples:", fruits.count('apple'))

# Find the index position of 'cherry'
print("Cherry index:", fruits.index('cherry'))
```

### `copy()`
If you type `list2 = list1`, changing `list1` will ALSO change `list2` because they share the same memory! Use `.copy()` to safely clone a list.
**Input:**
```python
fruits = ["apple", "banana"]
safe_copy = fruits.copy()

fruits.clear()
print("Safe Copy survived!", safe_copy)
```

### `reverse()`
Permanently reverses the current list.
**Input:**
```python
numbers = [1, 2, 3]
numbers.reverse()
print("Reversed:", numbers)
```

## 3. Deep Dive: Advanced Built-in Functions

### `min()`, `max()`, `sum()`
**Input:**
```python
nums = [10, 50, 5, 100]
print("Max:", max(nums))
print("Min:", min(nums))
print("Sum:", sum(nums))
```

### `sorted()` and `reversed()`
Unlike `.sort()` and `.reverse()`, these return BRAND NEW lists instead of modifying the original.
**Input:**
```python
original = [3, 1, 2]
new_sorted = sorted(original)
print("Original is safe:", original)
print("New Sorted:", new_sorted)

# reversed() returns an iterator, so we cast it to a list
new_reversed = list(reversed(original))
print("New Reversed:", new_reversed)
```

### `enumerate()`
Automatically generates the index number alongside the value in a `for` loop!
**Input:**
```python
fruits = ["apple", "banana", "cherry"]
for index, value in enumerate(fruits):
    print("Index:", index, "Value:", value)
```

### `zip()`
Combines two lists element-by-element like a zipper on a jacket!
**Input:**
```python
names = ["Alice", "Bob"]
ages = [25, 30]

zipped = list(zip(names, ages))
print("Zipped:", zipped)
```

### `map()` and `filter()`
Apply a function to every item, or filter items based on a function.
**Input:**
```python
def is_even(num):
    return num % 2 == 0

def double_it(num):
    return num * 2

numbers = [1, 2, 3, 4]

print("Mapped (Doubled):", list(map(double_it, numbers)))
print("Filtered (Evens):", list(filter(is_even, numbers)))
```
