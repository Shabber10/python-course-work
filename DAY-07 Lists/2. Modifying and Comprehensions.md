# Modifying and Comprehensions

Because lists are **mutable** (changeable), Python provides an enormous library of methods to add, remove, sort, and iterate through lists.

---

## 6. Changing List Items
To change the value of a specific item, refer to its index number. You can also change a range of items at once!

**Input:**
```python
fruits = ["apple", "banana", "cherry", "orange"]

# Change the second item
fruits[1] = "blackcurrant"
print("Changed Index 1:", fruits)

# Change a range of items (Replace 2 items with 2 new items)
fruits[1:3] = ["watermelon", "kiwi"]
print("Changed Slice:", fruits)
```
**Output:**
```text
Changed Index 1: ['apple', 'blackcurrant', 'cherry', 'orange']
Changed Slice: ['apple', 'watermelon', 'kiwi', 'orange']
```

---

## 7. Adding Items (`append`, `insert`, `extend`)
- `.append(item)`: Adds an item to the very end of the list.
- `.insert(index, item)`: Inserts an item at a specific index, pushing everything else to the right.
- `.extend(list2)`: Appends elements from another list to the current list.

**Input:**
```python
fruits = ["apple", "banana"]

# Append
fruits.append("orange")
print("After Append:", fruits)

# Insert at index 1
fruits.insert(1, "cherry")
print("After Insert:", fruits)

# Extend with another list
tropical = ["mango", "pineapple"]
fruits.extend(tropical)
print("After Extend:", fruits)
```
**Output:**
```text
After Append: ['apple', 'banana', 'orange']
After Insert: ['apple', 'cherry', 'banana', 'orange']
After Extend: ['apple', 'cherry', 'banana', 'orange', 'mango', 'pineapple']
```

---

## 8. Removing Items (`remove`, `pop`, `clear`)
- `.remove(item)`: Removes the first occurrence of the specified value.
- `.pop(index)`: Removes the specified index (if no index is provided, it removes the last item).
- `.clear()`: Empties the entire list, leaving `[]`.

**Input:**
```python
fruits = ["apple", "banana", "cherry", "banana"]

# Remove the first 'banana'
fruits.remove("banana")
print("After Remove:", fruits)

# Pop the last item
fruits.pop()
print("After Pop:", fruits)
```
**Output:**
```text
After Remove: ['apple', 'cherry', 'banana']
After Pop: ['apple', 'cherry']
```

## 9. List Comprehension (Advanced!)
List comprehension offers a shorter syntax when you want to create a new list based on the values of an existing list. It condenses a 4-line `for` loop into a single, elegant line of code.

Syntax: `newlist = [expression for item in iterable if condition == True]`

**Input:**
```python
fruits = ["apple", "banana", "cherry", "kiwi", "mango"]

# Goal: Create a list of fruits that contain the letter 'a'
# The OLD way (using a standard for loop):
newlist = []
for x in fruits:
    if "a" in x:
        newlist.append(x)
print("Old Way:", newlist)

# The NEW way (List Comprehension):
comp_list = [x for x in fruits if "a" in x]
print("Comprehension:", comp_list)
```
**Output:**
```text
Old Way: ['apple', 'banana', 'mango']
Comprehension: ['apple', 'banana', 'mango']
```

### A. Filtering and Transforming
You can combine filtering (using `if`) and transforming (using expressions) in a single comprehension.

**Input:**
```python
numbers = [1, 2, 3, 4, 5, 6]

# Goal: Square only the even numbers
squared_evens = [x**2 for x in numbers if x % 2 == 0]
print("Squared Evens:", squared_evens)
```
**Output:**
```text
Squared Evens: [4, 16, 36]
```

### B. List Comprehension with If-Else (Conditional Expressions)
If you want to return a different value depending on a condition, place the `if-else` statement **before** the `for` loop.

Syntax: `newlist = [value_if_true if condition else value_if_false for item in iterable]`

**Input:**
```python
numbers = [1, 2, 3, 4, 5]

# Label numbers as "Even" or "Odd"
labels = ["Even" if x % 2 == 0 else "Odd" for x in numbers]
print("Labels:", labels)
```
**Output:**
```text
Labels: ['Odd', 'Even', 'Odd', 'Even', 'Odd']
```

### C. Nested / 2D List Comprehensions
You can use a list comprehension inside another list comprehension to generate nested lists (like matrices/grids) or to flatten a 2D list.

**Input:**
```python
# 1. Generating a 3x3 coordinate matrix
matrix = [[(row, col) for col in range(3)] for row in range(3)]
print("Matrix:")
for row in matrix:
    print(row)

# 2. Flattening a 2D list into a single list
flat_list = [item for sublist in matrix for item in sublist]
print("Flattened Matrix (first 3 items):", flat_list[:3])
```
**Output:**
```text
Matrix:
[(0, 0), (0, 1), (0, 2)]
[(1, 0), (1, 1), (1, 2)]
[(2, 0), (2, 1), (2, 2)]
Flattened Matrix (first 3 items): [(0, 0), (0, 1), (0, 2)]
```

## 10. Sorting Lists
List objects have a `sort()` method that will sort the list alphanumerically, ascending, by default.

**Input:**
```python
cars = ["Ford", "BMW", "Volvo"]
cars.sort()
print("Alphabetical:", cars)

numbers = [100, 50, 65, 82, 23]
numbers.sort(reverse=True) # Sorts descending!
print("Descending:", numbers)
```
**Output:**
```text
Alphabetical: ['BMW', 'Ford', 'Volvo']
Descending: [100, 82, 65, 50, 23]
```
