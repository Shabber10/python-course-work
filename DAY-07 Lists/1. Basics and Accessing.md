# 1. Basics and Accessing

Lists are one of the 4 built-in data types in Python used to store collections of data. Lists are arguably the most powerful and commonly used data structure in all of Python.

---

## 1. List Characteristics
Lists are created using square brackets `[]`. They have four primary characteristics:

- **Ordered:** The items have a defined order, and that order will not change (unless you explicitly sort it).
- **Changeable (Mutable):** We can change, add, and remove items after the list has been created.
- **Allow Duplicates:** Because lists are indexed, they can have items with the exact same value.
- **Heterogeneous:** A single list can contain many different data types (strings, integers, and booleans all mixed together).

**Input:**
```python
# Creating a list with mixed data types and duplicates
mylist = ["apple", "banana", "cherry", "apple", 1, True]
print(mylist)

# Checking the length of the list
print("Length of list:", len(mylist))
```
**Output:**
```text
['apple', 'banana', 'cherry', 'apple', 1, True]
Length of list: 6
```

---

## 2. Accessing Items
List items are indexed. The first item has index `[0]`, the second item has index `[1]`, etc.

**Input:**
```python
fruits = ["apple", "banana", "cherry", "orange", "kiwi", "melon", "mango"]

# Access the second item
print("Index 1:", fruits[1])

# Negative Indexing (Starts from the end)
print("Index -1 (Last item):", fruits[-1])
```
**Output:**
```text
Index 1: banana
Index -1 (Last item): mango
```

---

## 3. Slicing Lists (Range of Indexes)
You can specify a range of indexes by specifying where to start and where to end the range. The return value will be a **brand new list** with the specified items.

Just like with strings, the search will start at index `start` (included) and end at index `end` (not included).

**Input:**
```python
fruits = ["apple", "banana", "cherry", "orange", "kiwi", "melon", "mango"]

# Return the third, fourth, and fifth items
print("Slice [2:5]:", fruits[2:5])

# From the start to the fourth item
print("Slice [:4]:", fruits[:4])

# From the third item to the end
print("Slice [2:]:", fruits[2:])
```
**Output:**
```text
Slice [2:5]: ['cherry', 'orange', 'kiwi']
Slice [:4]: ['apple', 'banana', 'cherry', 'orange']
Slice [2:]: ['cherry', 'orange', 'kiwi', 'melon', 'mango']
```

---

## 4. Checking if an Item Exists
To determine if a specified item is present in a list, use the `in` keyword. This is heavily used in conditional `if` statements!

**Input:**
```python
fruits = ["apple", "banana", "cherry"]

if "apple" in fruits:
    print("Yes, 'apple' is in the fruits list!")

if "orange" not in fruits:
    print("No, 'orange' is missing.")
```
**Output:**
```text
Yes, 'apple' is in the fruits list!
No, 'orange' is missing.
```

---

## 5. Simple Built-in Functions (`any` and `all`)
These two functions return a simple Boolean based on the contents of the list.

- `all()`: Returns `True` if **every single element** is True (or if the list is empty).
- `any()`: Returns `True` if **at least one element** is True.

*(Note: In Python, `0`, `False`, and empty strings `""` evaluate to False. Everything else is True!)*

**Input:**
```python
mixed_list = [0, 1, 1, 0]
true_list = [1, 1, True]

print("Any true in mixed_list?", any(mixed_list))
print("All true in mixed_list?", all(mixed_list))

print("All true in true_list?", all(true_list))
```
**Output:**
```text
Any true in mixed_list? True
All true in mixed_list? False
All true in true_list? True
```
