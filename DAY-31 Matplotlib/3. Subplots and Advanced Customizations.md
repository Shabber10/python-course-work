# 3. Subplots and Advanced Customizations

While Pyplot's state-based interface (`plt.plot()`, `plt.title()`) is excellent for simple, single charts, it becomes difficult to manage when you want to build dashboards with multiple panels or perform highly custom tweaks.

For complex layouts, Matplotlib provides a powerful **Object-Oriented (OO) Interface** using **Figures** and **Axes**.

---

## 1. The Object-Oriented Interface

Under the hood, Matplotlib views charts as a hierarchy:
- **`Figure` (Canvas)**: The overall window or page containing all drawing elements.
- **`Axes` (Subplots)**: The individual charts/plots themselves, containing labels, grid lines, and data lines. A single Figure can contain multiple Axes.

```text
  ┌────────────────────────────────────────────────────────┐
  │                    FIGURE (Canvas)                     │
  │                                                        │
  │   ┌────────────────────────┐  ┌────────────────────┐   │
  │   │      AXES 1 (Plot)     │  │    AXES 2 (Plot)   │   │
  │   │                        │  │                    │   │
  │   └────────────────────────┘  └────────────────────┘   │
  └────────────────────────────────────────────────────────┘
```

**Input:**
```python
import matplotlib.pyplot as plt

# Create Figure and a single Axes object
fig, ax = plt.subplots(figsize=(6, 4)) # figsize controls width and height in inches

x = [1, 2, 3, 4]
y = [10, 15, 7, 20]

# Plot directly on the axes object
ax.plot(x, y, color="purple", marker="o")

# In the OO interface, methods are prefixed with 'set_'
ax.set_title("Object-Oriented Line Plot")
ax.set_xlabel("X Axis Label")
ax.set_ylabel("Y Axis Label")
ax.grid(True)

plt.show()
```

---

## 2. Creating Subplots (Grid of Charts)

You can create grids of plots (dashboards) by passing the number of rows and columns to `plt.subplots()`. The resulting `axes` will be a NumPy array of Axes objects.

**Input:**
```python
import matplotlib.pyplot as plt

# Create a 1x2 grid (1 row, 2 columns) of plots
fig, axes = plt.subplots(nrows=1, ncols=2, figsize=(10, 4))

x = [1, 2, 3, 4]
y1 = [10, 20, 30, 40]
y2 = [40, 30, 20, 10]

# Access first subplot (index 0)
axes[0].plot(x, y1, color="blue", marker="o")
axes[0].set_title("First Subplot (Ascending)")

# Access second subplot (index 1)
axes[1].plot(x, y2, color="orange", marker="s", linestyle="--")
axes[1].set_title("Second Subplot (Descending)")

# Adjust layout to prevent overlaps between plots
plt.tight_layout()
plt.show()
```

---

## 3. Controlling Ticks, Limits, and Rotations

You can customize the range of the axes and control where numerical marks (ticks) appear.

**Input:**
```python
import matplotlib.pyplot as plt

fig, ax = plt.subplots(figsize=(6, 4))

months = ["January", "February", "March", "April"]
sales = [1000, 1500, 1200, 1800]

ax.plot(months, sales, marker="^", color="darkgreen")

# 1. Set axis range limits
ax.set_ylim(0, 2500)

# 2. Custom ticks
ax.set_yticks([0, 500, 1000, 1500, 2000, 2500])

# 3. Rotate x-axis label tickmarks to prevent overlapping labels
plt.setp(ax.get_xticklabels(), rotation=45, horizontalalignment="right")

ax.set_title("Monthly Sales Report")
ax.set_ylabel("Revenue ($)")
plt.tight_layout()
plt.show()
```

## 4. Annotating Data Points

Annotations allow you to point to specific elements in your chart and add explanatory text.

**Input:**
```python
import matplotlib.pyplot as plt

fig, ax = plt.subplots(figsize=(6, 4))

x = [1, 2, 3, 4, 5]
y = [10, 12, 25, 14, 18]

ax.plot(x, y, marker="o")

# Annotating the peak point (3, 25)
ax.annotate("Sales Peak!", 
            xy=(3, 25),          # Point to annotate (arrow head coordinate)
            xytext=(3.5, 22),    # Text placement coordinate
            arrowprops=dict(facecolor="black", shrink=0.05, width=1, headwidth=6))

ax.set_title("Product Sales Analytics")
ax.set_ylim(0, 30)
plt.show()
```
