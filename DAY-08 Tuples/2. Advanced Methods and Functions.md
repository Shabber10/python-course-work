# 2. Advanced Methods and Functions

Because tuples are immutable, they have significantly fewer methods than lists. You cannot add or remove items. However, they still have access to many powerful built-in analysis functions!

---

## 1. Master Reference Table
Here are all 2 Tuple Methods and 8 Built-in Functions that operate on Tuples.

### Tuple Methods
| Method | Description |
|---|---|
| `count()` | Returns the number of times a specified value occurs in a tuple. |
| `index()` | Searches the tuple for a specified value and returns the position of where it was found. |

### Built-in Functions for Tuples
| Function | Description |
|---|---|
| `len()` | Returns the length of the tuple. (Covered in Part 1) |
| `any()` | Returns True if any item is True. (Covered in Part 1) |
| `all()` | Returns True if all items are True. (Covered in Part 1) |
| `min()` | Returns the lowest value. |
| `max()` | Returns the highest value. |
| `sum()` | Returns the sum of all numerical items. |
| `tuple()` | Converts an iterable (like a list) into an immutable tuple. |
| `enumerate()`| Returns an enumerate object containing pairs of index/value. |
| `zip()` | Combines multiple tuples into a list of tuples. |

---

## 2. Deep Dive: Tuple Methods

### `count()` and `index()`
These are the ONLY two methods native to tuple objects.
**Input:**
```python
fruits = ('apple', 'banana', 'cherry', 'apple', 'apple')

# Count how many times 'apple' appears
print("Apples:", fruits.count('apple'))

# Find the index position of 'cherry'
print("Cherry index:", fruits.index('cherry'))
```

**Output:**
```text
Apples: 3
Cherry index: 2
```


---

## 3. Deep Dive: Built-in Functions

### `min()`, `max()`, `sum()`
**Input:**
```python
nums = (10, 50, 5, 100)
print("Max:", max(nums))
print("Min:", min(nums))
print("Sum:", sum(nums))
```

**Output:**
```text
Max: 100
Min: 5
Sum: 165
```


### The `tuple()` Constructor
The `tuple()` constructor function is used to convert other iterable data types (like Lists or Strings) directly into an immutable Tuple!

**Input:**
```python
# Convert a List to a Tuple
my_list = [1, 2, 3]
new_tuple = tuple(my_list)
print("List to Tuple:", new_tuple)

# Convert a String to a Tuple
my_string = "hello"
str_tuple = tuple(my_string)
print("String to Tuple:", str_tuple)
```

**Output:**
```text
List to Tuple: (1, 2, 3)
String to Tuple: ('h', 'e', 'l', 'l', 'o')
```


### `enumerate()`
Automatically generates the index number alongside the value in a `for` loop!
**Input:**
```python
fruits = ("apple", "banana", "cherry")
for index, value in enumerate(fruits):
    print("Index:", index, "Value:", value)
```

**Output:**
```text
Index: 0 Value: apple
Index: 1 Value: banana
Index: 2 Value: cherry
```


### `zip()`
Combines two tuples element-by-element!
**Input:**
```python
names = ("Alice", "Bob")
ages = (25, 30)

zipped = tuple(zip(names, ages))
print("Zipped:", zipped)
```

**Output:**
```text
Zipped: (('Alice', 25), ('Bob', 30))
```
