# 1. Conditionals (If, Elif, Else)

Python supports the usual logical conditions from mathematics. These conditions can be used in several ways, most commonly in "if statements" and loops.

---

## 1. The `if` Statement
An `if` statement is written using the `if` keyword. The code block inside the `if` statement will ONLY execute if the condition evaluates to `True`.

**Remember:** Python relies on **indentation** (whitespace at the beginning of a line) to define scope in the code.

**Input:**
```python
a = 33
b = 200

if b > a:
    print("b is strictly greater than a")
```
**Output:**
```text
b is strictly greater than a
```

### Indentation Error
If you do not indent the code inside the `if` statement, Python will crash!
**Input:**
```python
a = 33
b = 200
# if b > a:
# print("b is greater than a") # ERROR: expected an indented block!
```

---

## 2. The `elif` Statement
The `elif` keyword is Python's way of saying *"if the previous conditions were not true, then try this condition"*. 

**Input:**
```python
a = 33
b = 33

if b > a:
    print("b is greater than a")
elif a == b:
    print("a and b are exactly equal")
```
**Output:**
```text
a and b are exactly equal
```

---

## 3. The `else` Statement
The `else` keyword catches anything which isn't caught by the preceding conditions. It acts as the final "fallback".

**Input:**
```python
a = 200
b = 33

if b > a:
    print("b is greater than a")
elif a == b:
    print("a and b are equal")
else:
    print("a must be greater than b")
```
**Output:**
```text
a must be greater than b
```

---

## 4. Logical Operators in Conditionals
You can combine multiple conditions into a single `if` statement using `and`, `or`, and `not`.

**Input:**
```python
a = 200
b = 33
c = 500

# Both conditions must be True!
if a > b and c > a:
    print("Both conditions are True")

# Only ONE condition must be True!
if a > b or a > c:
    print("At least one of the conditions is True")

# Using NOT
if not a == b:
    print("a is NOT equal to b")
```
**Output:**
```text
Both conditions are True
At least one of the conditions is True
a is NOT equal to b
```
