# 1. Introduction to Control Statements

Control statements alter the flow of execution in a Python program. By default, a Python script executes top-to-bottom, line by line. Control statements allow you to skip lines, repeat lines, or branch into different paths based on conditions.

Python provides three main types of control statements:

---

## 1. Conditional Statements
Used to execute specific blocks of code only if a certain condition is met. These are the decision-making statements in Python (`if`, `elif`, `else`).

**Input:**
```python
age = 18

if age >= 18:
    print("You are eligible to vote!")
else:
    print("You are too young to vote.")
```
**Output:**
```text
You are eligible to vote!
```

---

## 2. Looping Statements
Used to repeatedly execute a block of code as long as a condition is true, or to iterate over a sequence of items (`while`, `for`).

**Input:**
```python
# Example of a For Loop repeating an action
print("Counting to 3:")
for i in range(1, 4):
    print(i)
```
**Output:**
```text
Counting to 3:
1
2
3
```

---

## 3. Jumping Statements
Used to alter the flow of loops unconditionally. They allow you to exit a loop entirely, skip an iteration, or serve as a placeholder (`break`, `continue`, `pass`).

**Input:**
```python
# Example of breaking out of a loop early
for number in range(1, 6):
    if number == 4:
        break # Stops the loop when number hits 4
    print("Number is:", number)
```
**Output:**
```text
Number is: 1
Number is: 2
Number is: 3
```
