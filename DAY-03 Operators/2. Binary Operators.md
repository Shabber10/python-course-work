# 2. Binary Operators

Operate on two operands. Most operators in Python are binary.

## 2.1 Arithmetic Operators
Arithmetic operators are used with numeric values to perform common mathematical operations.

**Input:**
```python
a = 10
b = 3
print(a + b)  # Addition (13)
print(a - b)  # Subtraction (7)
print(a * b)  # Multiplication (30)
print(a / b)  # Normal division (3.3333333333333335)
print(a // b) # Floor division (3)
print(a % b)  # Modulus (1)
print(a ** b) # Power / Exponentiation (1000)
```

## 2.2 Assignment Operators
Used to assign values to variables (`=`, `+=`, `-=`, `*=`, `/=`).
**Input:**
```python
x = 5
x += 3 # Equivalent to x = x + 3
print(x) # Output: 8
```

## 2.3 Comparison (Relational) Operators
Used to compare two values. They return `True` or `False`. (`==`, `!=`, `>`, `<`, `>=`, `<=`)
**Input:**
```python
print(5 > 3)  # Output: True
print(5 == 3) # Output: False
```

## 2.4 Logical Operators
Used to combine conditional statements (`and`, `or`, `not`).
**Input:**
```python
x = 5
print(x > 3 and x < 10)     # Output: True
print(not(x > 3 and x < 10)) # Output: False
```

## 2.5 Identity Operators
Used to compare the objects (memory locations), not if they are equal, but if they are actually the same object.
**Input:**
```python
x = ["apple", "banana"]
y = ["apple", "banana"]
z = x
print(x is z) # Same object -> Output: True
print(x is y) # Different objects, same values -> Output: False
print(x == y) # Values are equal -> Output: True
```

## 2.6 Membership Operators
Used to test if a sequence is presented in an object.
**Input:**
```python
fruits = ["apple", "banana"]
print("banana" in fruits)      # Output: True
print("pineapple" not in fruits) # Output: True
```

## 2.7 Bitwise Operators
Used to compare (binary) numbers (`&`, `|`, `^`, `~`, `<<`, `>>`).
**Input:**
```python
# 6 = 0110 in binary
# 3 = 0011 in binary
print(6 & 3) # AND: 0010 (2)
print(6 | 3) # OR:  0111 (7)
```
