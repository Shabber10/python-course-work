# 4. Input and Output Functions

In Python, programs often need to interact with the user. The two most fundamental ways to achieve this are by taking input from the user via the keyboard, and displaying output back to the screen. 

---

## 1. The Output Function: `print()`

The `print()` function is used to display output to the console. It is one of the most commonly used functions in Python.

### A. Basic Printing
You can print strings, numbers, variables, and the results of calculations.

**Input:**
```python
print("Welcome to Python!")
print(100)
print(50 + 50)
```
**Output:**
```text
Welcome to Python!
100
100
```

### B. Printing Multiple Values
You can pass multiple items to `print()` by separating them with commas. By default, Python adds a space between them.

**Input:**
```python
name = "Alice"
age = 25
print("My name is", name, "and I am", age, "years old.")
```
**Output:**
```text
My name is Alice and I am 25 years old.
```

### C. Advanced `print()` formatting (`sep` and `end`)
The `print()` function has two powerful, optional parameters:
- `sep`: Defines what character to put between multiple items (default is a space `" "`).
- `end`: Defines what to print at the very end (default is a newline `\n`).

**Input:**
```python
# Using sep to separate items with a dash
print("apple", "banana", "cherry", sep="-")

# Using end to prevent the print function from creating a new line
print("Loading", end="...")
print("Complete!")
```
**Output:**
```text
apple-banana-cherry
Loading...Complete!
```

---

## 2. The Input Function: `input()`

The `input()` function pauses the program and waits for the user to type something on their keyboard and press Enter. 

### A. Basic Input
Whatever the user types is **always returned as a String**, even if they type a number!

**Input:**
```python
name = input("Please enter your name: ")
print("Hello,", name)
```
*(User types: Shabber)*
**Output:**
```text
Please enter your name: Shabber
Hello, Shabber
```

### B. Casting Input to Numbers
Because `input()` always gives you a string, if you want to do math with the user's input, you MUST use explicit type casting (like `int()` or `float()`) to convert their string into a number.

**Input:**
```python
# If you don't cast to an integer, "10" + "5" would become "105"
age_str = input("Enter your age: ")
age_int = int(age_str)

print("In 5 years, you will be:", age_int + 5)
```
*(User types: 20)*
**Output:**
```text
Enter your age: 20
In 5 years, you will be: 25
```

### C. Best Practices (Inline Casting)
Professional Python developers usually wrap the `input()` function directly inside the casting function to save lines of code.

**Input:**
```python
price = float(input("Enter the item price: $"))
tax = price * 0.08
print("The tax is: $", tax)
```
*(User types: 50.00)*
**Output:**
```text
Enter the item price: $50.00
The tax is: $ 4.0
```
