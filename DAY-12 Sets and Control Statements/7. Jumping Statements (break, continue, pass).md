### 3. Jumping Statements (`break`, `continue`, `pass`)
Used to alter the flow of loops or stub out code.
**Input:**
```python
for i in range(5):
    if i == 1:
        continue # Skip iteration 1
    if i == 3:
        break    # Stop the loop at 3
    print(i)

# pass does nothing, used as placeholder
def my_function():
    pass
```
**Output:**
```text
0
2
```
