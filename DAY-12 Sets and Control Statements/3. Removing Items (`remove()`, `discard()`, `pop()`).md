### 3. Removing Items (`remove()`, `discard()`, `pop()`)
**Input:**
```python
thisset = {"apple", "banana", "cherry"}

# Remove raises an error if item doesn't exist
thisset.remove("banana")
print(thisset)

# Discard does NOT raise an error if item doesn't exist
thisset.discard("cherry")
print(thisset)

# Pop removes a random item
x = thisset.pop()
print("Popped:", x)
print("Remaining:", thisset)
```
**Output:**
```text
{'cherry', 'apple'}
{'apple'}
Popped: apple
Remaining: set()
```
