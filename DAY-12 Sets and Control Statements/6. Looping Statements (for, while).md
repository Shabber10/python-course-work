### 2. Looping Statements (`for`, `while`)
Used to repeat a block of code.
**Input:**
```python
# For Loop
for i in range(2):
    print("For loop:", i)

# While Loop
count = 0
while count < 2:
    print("While loop:", count)
    count += 1
```
**Output:**
```text
For loop: 0
For loop: 1
While loop: 0
While loop: 1
```
