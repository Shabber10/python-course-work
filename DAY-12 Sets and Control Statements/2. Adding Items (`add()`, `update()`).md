### 2. Adding Items (`add()`, `update()`)
**Input:**
```python
thisset = {"apple", "banana"}

# Add a single item
thisset.add("cherry")
print(thisset)

# Add multiple items from another set (or list)
tropical = {"pineapple", "mango"}
thisset.update(tropical)
print(thisset)
```
**Output:**
```text
{'banana', 'cherry', 'apple'}
{'banana', 'cherry', 'pineapple', 'apple', 'mango'}
```
