### 4. Set Operations (Union, Intersection, Difference)
**Input:**
```python
set1 = {"a", "b", "c"}
set2 = {"c", "d", "e"}

# Union (|) - Combines both sets
print(set1 | set2)

# Intersection (&) - Keeps only items present in both
print(set1 & set2)

# Difference (-) - Keeps items only in set1
print(set1 - set2)

# Symmetric Difference (^) - Keeps all except duplicates
print(set1 ^ set2)
```
**Output:**
```text
{'d', 'c', 'b', 'e', 'a'}
{'c'}
{'b', 'a'}
{'d', 'b', 'e', 'a'}
```
