## 2. Replacing and Splitting

### `replace()`
Replaces a string with another string.
**Input:**
```python
a = "Hello, World!"
print(a.replace("H", "J"))
```
**Output:**
```text
Jello, World!
```

### `split()`
Splits the string into substrings if it finds instances of the separator.
**Input:**
```python
a = "Hello, World!"
print(a.split(","))
```
**Output:**
```text
['Hello', ' World!']
```
