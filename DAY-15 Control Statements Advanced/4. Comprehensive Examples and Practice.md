## Example 1: Finding Prime Numbers in a Range
This example uses nested loops, conditional statements, and the `for...else` suite.

```python
lower_bound = 10
upper_bound = 30

print(f"Prime numbers between {lower_bound} and {upper_bound}:")

for num in range(lower_bound, upper_bound + 1):
    # Prime numbers are greater than 1
    if num > 1:
        for i in range(2, int(num**0.5) + 1):
            if (num % i) == 0:
                # If a factor is found, it's not prime. Break out of the inner loop.
                break
        else:
            # This else block executes if the inner loop completes without a break.
            # Which means no factors were found, so the number is prime.
            print(num, end=" ")
print()
```
**Output:**
`11 13 17 19 23 29`

## Example 2: Simple ATM Menu System
This example uses a `while True` infinite loop, `match-case` (or `if-elif` if < Python 3.10), and `break` to exit the loop.

```python
balance = 1000.0

while True:
    print("\n--- ATM Menu ---")
    print("1. Check Balance")
    print("2. Deposit Money")
    print("3. Withdraw Money")
    print("4. Exit")
    
    choice = input("Enter your choice (1-4): ")
    
    match choice:
        case "1":
            print(f"Your current balance is: ${balance:.2f}")
        case "2":
            amount = float(input("Enter amount to deposit: $"))
            if amount > 0:
                balance += amount
                print(f"Successfully deposited ${amount:.2f}.")
            else:
                print("Invalid deposit amount.")
        case "3":
            amount = float(input("Enter amount to withdraw: $"))
            if 0 < amount <= balance:
                balance -= amount
                print(f"Successfully withdrew ${amount:.2f}.")
            elif amount > balance:
                print("Insufficient funds.")
            else:
                print("Invalid withdrawal amount.")
        case "4":
            print("Thank you for using the ATM. Goodbye!")
            break  # Exits the infinite while loop
        case _:
            print("Invalid choice. Please select a valid option.")
```

## Practice Problems

Try writing code for the following scenarios to test your understanding:

### Problem 1: Number Guessing Game
Write a program where the computer generates a random number between 1 and 50. Give the user 5 attempts to guess the number.
- Use a `while` loop to limit attempts.
- Provide hints like "Too high!" or "Too low!".
- Use `break` if they guess correctly.
- Use the `else` block on the `while` loop to print "Game Over! You've run out of attempts." if they fail all 5 times.

### Problem 2: Password Validator
Ask the user to enter a password. Keep asking them using a `while True` loop until they enter a valid password.
Conditions for a valid password:
- At least 8 characters long.
- Contains the word "admin" (for this simple example).
If they enter a correct password, `break` the loop and print "Access Granted". If not, use `continue` to skip to the next iteration.
