# 3. The Math Module

Python's built-in **`math`** module provides access to the mathematical functions defined by the C standard. It is essential for performing advanced arithmetic, algebraic, trigonometric, and rounding calculations.

---

## 1. Mathematical Constants
The `math` module exposes several mathematical constants.

**Input:**
```python
import math

print("Pi (Ď€):", math.pi)
print("Euler's number (e):", math.e)
```
**Output:**
```text
Pi (Ď€): 3.141592653589793
Euler's number (e): 2.718281828459045
```

---

## 2. Basic Arithmetic and Algebraic Methods

### A. Square Root and Powers: `math.sqrt()` and `math.pow()`
- `math.sqrt(x)`: Returns the square root of $x$ as a float.
- `math.pow(x, y)`: Returns $x$ raised to the power $y$ ($x^y$) as a float.

**Input:**
```python
print("Square root of 25:", math.sqrt(25))
print("2 raised to power 5:", math.pow(2, 5))
```
**Output:**
```text
Square root of 25: 5.0
2 raised to power 5: 32.0
```

### B. Factorial: `math.factorial(n)`
Returns $n!$ (factorial of $n$). Raises a `ValueError` if $n$ is negative or not an integer.

**Input:**
```python
print("Factorial of 5:", math.factorial(5))
```
**Output:**
```text
Factorial of 5: 120
```

### C. Greatest Common Divisor: `math.gcd(a, b)`
Returns the greatest common divisor of the integers $a$ and $b$.

**Input:**
```python
print("GCD of 25 and 80:", math.gcd(25, 80))
```
**Output:**
```text
GCD of 25 and 80: 5
```

### D. Logarithm: `math.log(x, [base])`
With one argument, returns the natural logarithm of $x$ (to base $e$). With two arguments, returns the logarithm of $x$ to the given $base$.

**Input:**
```python
# Logarithm of 100 to base 10
print("Log of 100 to base 10:", math.log(100, 10))
```
**Output:**
```text
Log of 100 to base 10: 2.0
```

---

## 3. Trigonometric and Angle Conversion Methods
Trigonometric functions in Python expect angles in **radians**, not degrees.

- `math.sin(x)`, `math.cos(x)`, `math.tan(x)`: Trigonometric functions.
- `math.degrees(x)`: Converts angle $x$ from radians to degrees.
- `math.radians(x)`: Converts angle $x$ from degrees to radians.

**Input:**
```python
# Convert 90 degrees to radians
rad = math.radians(90)
print("90 degrees in radians:", rad)

# Find sine of 90 degrees (which is sin(1.57079... radians))
print("sin(90 degrees):", math.sin(rad))

# Convert radians back to degrees
deg = math.degrees(rad)
print("Radians converted back to degrees:", deg)
```
**Output:**
```text
90 degrees in radians: 1.5707963267948966
sin(90 degrees): 1.0
Radians converted back to degrees: 90.0
```

---

## 4. Rounding Methods

| Method | Behavior | Example |
| :--- | :--- | :--- |
| **`round(x, [n])`** | Rounds to nearest value; rounds to even on exact halves (built-in). | `round(2.5) -> 2`, `round(3.5) -> 4` |
| **`math.ceil(x)`** | Rounds up to the nearest integer (returns smallest integer $\ge x$). | `math.ceil(2.1) -> 3` |
| **`math.floor(x)`** | Rounds down to the nearest integer (returns largest integer $\le x$). | `math.floor(2.9) -> 2` |

**Input:**
```python
print("round(25.550):", round(25.550))
print("math.ceil(2.123):", math.ceil(2.123))
print("math.floor(258.966):", math.floor(258.966))
```
**Output:**
```text
round(25.550): 26
math.ceil(2.123): 3
math.floor(258.966): 258
```
