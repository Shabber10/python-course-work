# 1. Random and Time Modules

Python comes with a rich set of built-in libraries. Two of the most commonly used modules for handling probabilities and system timings are `random` and `time`.

---

## 1. The `random` Module
The `random` module provides tools to generate pseudo-random numbers and make random selections from collections.

### A. Generating Random Floats: `random.random()`
Returns a random floating-point number in the range `[0.0, 1.0)`.

**Input:**
```python
import random

# Generate a float between 0.0 and 1.0
print("Random float:", random.random())
```
**Output:**
```text
Random float: 0.3745401188473625
```

### B. Generating Random Integers: `random.randint(a, b)`
Returns a random integer $N$ such that $a \le N \le b$ (both limits are inclusive).

**Input:**
```python
# Roll a 6-sided die
die_roll = random.randint(1, 6)
print("Die roll:", die_roll)
```
**Output:**
```text
Die roll: 5
```

### C. Making Random Selections: `random.choice(seq)`
Returns a random element from a non-empty sequence (like a list or tuple).

**Input:**
```python
fruits = ["apple", "banana", "cherry", "date"]
picked_fruit = random.choice(fruits)
print("Randomly picked fruit:", picked_fruit)
```
**Output:**
```text
Randomly picked fruit: banana
```

### D. Shuffling in Place: `random.shuffle(list)`
Shuffles the elements of a list in place (mutates the list).

**Input:**
```python
cards = ["Ace", "King", "Queen", "Jack"]
random.shuffle(cards)
print("Shuffled cards:", cards)
```
**Output:**
```text
Shuffled cards: ['Queen', 'Jack', 'Ace', 'King']
```

---

## 2. The `time` Module
The `time` module provides functions for working with time-related tasks.

### A. Unix Epoch Time: `time.time()`
Returns the current system time in seconds since the Epoch (January 1, 1970, 00:00:00 UTC). It is useful for measuring code execution speed.

**Input:**
```python
import time

epoch_time = time.time()
print("Seconds since Epoch:", epoch_time)
```
**Output:**
```text
Seconds since Epoch: 1782555600.0
```

### B. Execution Delay: `time.sleep(seconds)`
Suspends execution of the current thread for the given number of seconds.

**Input:**
```python
print("Starting...")
time.sleep(2)  # Pauses execution for exactly 2 seconds
print("Completed after 2 seconds delay!")
```
**Output:**
```text
Starting...
Completed after 2 seconds delay!
```
