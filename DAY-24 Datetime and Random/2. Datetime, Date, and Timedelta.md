# 2. Datetime, Date, and Timedelta

While the `time` module works with raw seconds, the `datetime` module offers high-level, object-oriented tools to manipulate calendar dates, times, intervals, and timezones.

---

## 1. Key Classes in `datetime`
The module contains three main classes for managing timestamps:
1. `date`: Represents a calendar date (Year, Month, Day).
2. `time`: Represents a time of day (Hour, Minute, Second, Microsecond).
3. `datetime`: Combines both date and time values.

### Getting Current Date & Time:
**Input:**
```python
from datetime import date, datetime

# Today's date
today = date.today()
print("Today's date:", today)
print(f"Year: {today.year}, Month: {today.month}, Day: {today.day}")

# Current date and time
now = datetime.now()
print("Current date & time:", now)
```
**Output:**
```text
Today's date: 2026-06-27
Year: 2026, Month: 6, Day: 27
Current date & time: 2026-06-27 15:30:00.123456
```

---

## 2. Formatting & Parsing Dates

### A. Formatting Object to String (`strftime`)
`strftime` (string format time) converts a date/datetime object into a customized, readable string format.

Common format codes:
- `%Y`: Year (4 digits)
- `%m`: Month (2 digits)
- `%d`: Day of the month (2 digits)
- `%H`: Hour (24-hour clock)
- `%M`: Minute
- `%S`: Second
- `%A`: Full weekday name

**Input:**
```python
now = datetime(2026, 6, 27, 15, 30, 0)

# Format: YYYY-MM-DD HH:MM:SS
formatted_str1 = now.strftime("%Y-%m-%d %H:%M:%S")
print("Format 1:", formatted_str1)

# Format: Day Name, Month, Year
formatted_str2 = now.strftime("%A, %d %B %Y")
print("Format 2:", formatted_str2)
```
**Output:**
```text
Format 1: 2026-06-27 15:30:00
Format 2: Saturday, 27 June 2026
```

### B. Parsing String to Object (`strptime`)
`strptime` (string parse time) converts a date string into a Python datetime object.

**Input:**
```python
date_string = "27-06-2026 15:30"
format_pattern = "%d-%m-%Y %H:%M"

# Parsing
parsed_datetime = datetime.strptime(date_string, format_pattern)
print("Parsed datetime object:", parsed_datetime)
print("Type:", type(parsed_datetime))
```
**Output:**
```text
Parsed datetime object: 2026-06-27 15:30:00
Type: <class 'datetime.datetime'>
```

---

## 3. Date Arithmetic: `timedelta`
A `timedelta` object represents a duration—the difference between two dates or times. You can add or subtract `timedelta` objects from `date` or `datetime` objects to perform date calculations.

### A. Finding Difference Between Dates
**Input:**
```python
date1 = date(2026, 6, 27)
date2 = date(2026, 7, 7)

# Difference returns a timedelta object
difference = date2 - date1
print("Difference:", difference)
print("Days between:", difference.days)
```
**Output:**
```text
Difference: 10 days, 0:00:00
Days between: 10
```

### B. Adding / Subtracting Days or Weeks
**Input:**
```python
from datetime import timedelta

today = date(2026, 6, 27)

# Calculate date 10 days in the future
future_date = today + timedelta(days=10)
print("Date after 10 days:", future_date)

# Calculate date 3 weeks in the past
past_date = today - timedelta(weeks=3)
print("Date 3 weeks ago:", past_date)
```
**Output:**
```text
Date after 10 days: 2026-07-07
Date 3 weeks ago: 2026-06-06
```
