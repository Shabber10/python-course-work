# 2. Datetime, Date, and Timedelta

While the `time` module works with raw seconds, the `datetime` module offers high-level, object-oriented tools to manipulate calendar dates, times, intervals, and timezones.

---

## 1. Key Classes in `datetime`
The module contains three main classes for managing timestamps:
1. `date`: Represents a calendar date (Year, Month, Day).
2. `time`: Represents a time of day (Hour, Minute, Second, Microsecond).
3. `datetime`: Combines both date and time values.

### Getting Current Date & Time:
**Input:**
```python
from datetime import date, datetime

# Today's date
today = date.today()
print("Today's date:", today)
print(f"Year: {today.year}, Month: {today.month}, Day: {today.day}")

# Current date and time
now = datetime.now()
print("Current date & time:", now)
```
**Output:**
```text
Today's date: 2026-06-27
Year: 2026, Month: 6, Day: 27
Current date & time: 2026-06-27 15:30:00.123456
```

### B. Finding the Day of the Week
Python provides two methods to get the day of the week:
- **`date.weekday()`**: Returns the day of the week as an integer where **Monday is 0 and Sunday is 6**.
- **`date.isoweekday()`**: Returns the day of the week as an integer where **Monday is 1 and Sunday is 7**.

**Input:**
```python
dt = date(2026, 6, 27)  # This is a Saturday

print("date.weekday() (Monday=0):", dt.weekday())
print("date.isoweekday() (Monday=1):", dt.isoweekday())
```
**Output:**
```text
date.weekday() (Monday=0): 5
date.isoweekday() (Monday=1): 6
```

## 2. Formatting & Parsing Dates

### A. Formatting Object to String (`strftime`)
`strftime` (string format time) converts a date/datetime object into a customized, readable string format.

### B. Parsing String to Object (`strptime`)
`strptime` (string parse time) converts a date string into a Python datetime object.

### 📊 Common Date Formatting Codes Cheat Sheet
Use these format codes inside `strftime` and `strptime`:

| Code | Meaning | Example |
| :---: | :--- | :--- |
| **`%Y`** | Year (4 digits) | `2026` |
| **`%y`** | Year (2 digits, zero-padded) | `26` |
| **`%m`** | Month (2 digits, zero-padded) | `06` |
| **`%B`** | Month name (Full) | `June` |
| **`%b`** | Month name (Abbreviated) | `Jun` |
| **`%d`** | Day of the Month (zero-padded) | `27` |
| **`%A`** | Weekday name (Full) | `Saturday` |
| **`%H`** | Hour (24-hour clock, zero-padded) | `15` |
| **`%I`** | Hour (12-hour clock, zero-padded) | `03` |
| **`%p`** | AM or PM indicator | `PM` |
| **`%M`** | Minute (zero-padded) | `30` |
| **`%S`** | Second (zero-padded) | `00` |

**Input:**
```python
now = datetime(2026, 6, 27, 15, 30, 0)

# Format: YYYY-MM-DD HH:MM:SS
formatted_str1 = now.strftime("%Y-%m-%d %H:%M:%S")
print("Format 1:", formatted_str1)

# Format: Day Name, Month, Year
formatted_str2 = now.strftime("%A, %d %B %Y")
print("Format 2:", formatted_str2)

# Parsing back to object
date_string = "27-06-2026 15:30"
parsed_datetime = datetime.strptime(date_string, "%d-%m-%Y %H:%M")
print("Parsed Object:", parsed_datetime)
```
**Output:**
```text
Format 1: 2026-06-27 15:30:00
Format 2: Saturday, 27 June 2026
Parsed Object: 2026-06-27 15:30:00
```

## 3. Date Arithmetic: `timedelta`
A `timedelta` object represents a duration—the difference between two dates or times. You can add or subtract `timedelta` objects from `date` or `datetime` objects to perform date calculations.

### A. Finding Difference Between Dates
**Input:**
```python
date1 = date(2026, 6, 27)
date2 = date(2026, 7, 7)

# Difference returns a timedelta object
difference = date2 - date1
print("Difference:", difference)
print("Days between:", difference.days)
```
**Output:**
```text
Difference: 10 days, 0:00:00
Days between: 10
```

### B. Adding / Subtracting Days or Weeks
**Input:**
```python
from datetime import timedelta

today = date(2026, 6, 27)

# Calculate date 10 days in the future
future_date = today + timedelta(days=10)
print("Date after 10 days:", future_date)

# Calculate date 3 weeks in the past
past_date = today - timedelta(weeks=3)
print("Date 3 weeks ago:", past_date)
```
**Output:**
```text
Date after 10 days: 2026-07-07
Date 3 weeks ago: 2026-06-06
```

### C. Adding / Subtracting Time (Hours, Minutes, Seconds)
`timedelta` can also store time differences down to microseconds. This is useful for adjusting `datetime` objects.

**Input:**
```python
now = datetime(2026, 6, 27, 15, 30, 0)

# 2 hours in the future
future_time = now + timedelta(hours=2)
print("Time after 2 hours:", future_time)

# 15 minutes in the past
past_time = now - timedelta(minutes=15)
print("Time 15 mins ago:", past_time)

# 15 seconds in the future
future_secs = now + timedelta(seconds=15)
print("Time after 15 secs:", future_secs)
```
**Output:**
```text
Time after 2 hours: 2026-06-27 17:30:00
Time 15 mins ago: 2026-06-27 15:15:00
Time after 15 secs: 2026-06-27 15:30:15
```

## 4. Timezone Naive vs. Timezone Aware
By default, datetime objects in Python are **naive**, meaning they do not hold any information about timezone regions (like UTC or EST). 

If you are writing programs that run across different physical servers or serve global users, you must make your datetimes **timezone aware**.

**Input:**
```python
from datetime import timezone

# Naive datetime (no timezone info)
naive_dt = datetime.now()
print("Naive timezone info:", naive_dt.tzinfo)

# Aware datetime (using UTC timezone)
aware_dt = datetime.now(timezone.utc)
print("Aware timezone info:", aware_dt.tzinfo)
print("Aware Datetime:", aware_dt)
```
**Output:**
```text
Naive timezone info: None
Aware timezone info: UTC
Aware Datetime: 2026-06-27 10:00:00.123456+00:00
```
