# 4. Ternary Operator

Operates on three operands. In Python, this is used for conditional expressions (shorthand `if-else`).

**Input:**
```python
age = 20
status = "Adult" if age >= 18 else "Minor"
print(status)
```
**Output:**
```text
Adult
```
