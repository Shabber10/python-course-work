# 2. Binary Arithmetic and Assignment

Binary operators are operators that operate on **two operands** (values or variables). Python provides several types of binary operators. In this part, we will explore Arithmetic and Assignment operators in extreme detail.

---

## 1. Arithmetic Operators

Arithmetic operators are used with numeric values to perform common mathematical operations such as addition, subtraction, multiplication, and division.

### A. Addition (`+`) and Subtraction (`-`)
The addition operator adds two values, while the subtraction operator subtracts the right operand from the left operand.

**Input:**
```python
x = 15
y = 4

print("Addition (x + y):", x + y)
print("Subtraction (x - y):", x - y)
```
**Output:**
```text
Addition (x + y): 19
Subtraction (x - y): 11
```

### B. Multiplication (`*`) and Division (`/`)
The multiplication operator multiplies two values. The division operator divides the left operand by the right operand, and **always returns a float**, even if the division is perfectly clean.

**Input:**
```python
a = 10
b = 2

print("Multiplication (a * b):", a * b)
print("Division (a / b):", a / b) # Notice this prints 5.0, not 5
```
**Output:**
```text
Multiplication (a * b): 20
Division (a / b): 5.0
```

### C. Floor Division (`//`)
Floor division divides the left operand by the right operand and rounds the result **down** to the nearest whole number (integer).

**Input:**
```python
print("Normal Division (15 / 4):", 15 / 4)
print("Floor Division (15 // 4):", 15 // 4) # Rounds 3.75 down to 3

# Negative floor division rounds away from zero!
print("Negative Floor (-15 // 4):", -15 // 4) # Rounds -3.75 down to -4
```
**Output:**
```text
Normal Division (15 / 4): 3.75
Floor Division (15 // 4): 3
Negative Floor (-15 // 4): -4
```

### D. Modulus (`%`)
The modulus operator returns the **remainder** of the division. This is extremely useful for checking if a number is even or odd!

**Input:**
```python
print("Remainder of 10 / 3 is:", 10 % 3)
print("Remainder of 10 / 2 is:", 10 % 2) # Even numbers always have remainder 0

# Checking Even or Odd
number = 7
if number % 2 == 0:
    print(number, "is Even")
else:
    print(number, "is Odd")
```
**Output:**
```text
Remainder of 10 / 3 is: 1
Remainder of 10 / 2 is: 0
7 is Odd
```

### E. Exponentiation (`**`)
The exponentiation operator raises the left operand to the power of the right operand (e.g., $x^y$).

**Input:**
```python
base = 2
power = 3
print("2 to the power of 3 is:", base ** power) # 2 * 2 * 2

# You can also do fractional powers! (e.g., square root)
print("Square root of 16 is:", 16 ** 0.5)
```
**Output:**
```text
2 to the power of 3 is: 8
Square root of 16 is: 4.0
```

---

## 2. Assignment Operators

Assignment operators are used to assign values to variables.

### A. Simple Assignment (`=`)
Assigns the value on the right side to the variable on the left side.

**Input:**
```python
x = 5
print("x is:", x)
```
**Output:**
```text
x is: 5
```

### B. Compound Assignment Operators
These operators combine an arithmetic operation with the assignment. It's a shorthand way of updating a variable's value!

#### Addition Assignment (`+=`)
`x += 3` is exactly the same as `x = x + 3`.

**Input:**
```python
x = 10
print("Original x:", x)

x += 5 # Adds 5 to the current value of x
print("After x += 5:", x)
```
**Output:**
```text
Original x: 10
After x += 5: 15
```

#### Subtraction Assignment (`-=`)
`x -= 3` is exactly the same as `x = x - 3`.

**Input:**
```python
y = 20
y -= 5
print("After y -= 5:", y)
```
**Output:**
```text
After y -= 5: 15
```

#### Multiplication Assignment (`*=`)
`x *= 3` is exactly the same as `x = x * 3`.

**Input:**
```python
z = 4
z *= 3
print("After z *= 3:", z)
```
**Output:**
```text
After z *= 3: 12
```

#### Division Assignment (`/=`)
`x /= 3` is exactly the same as `x = x / 3`.

**Input:**
```python
a = 15
a /= 3
print("After a /= 3:", a)
```
**Output:**
```text
After a /= 3: 5.0
```

#### Other Compound Operators
You can use compound assignment for every arithmetic operator!
- `x %= 3`
- `x //= 3`
- `x **= 3`

**Input:**
```python
power_val = 5
power_val **= 2 # Same as power_val = 5 ** 2
print("After power_val **= 2:", power_val)
```
**Output:**
```text
After power_val **= 2: 25
```
