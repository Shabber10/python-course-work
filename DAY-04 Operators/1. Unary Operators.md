# 1. Unary Operators

Operate on a single operand.
- Unary plus `+a`
- Unary minus `-a`
- Bitwise NOT `~a`
- Logical NOT `not a`

**Input:**
```python
x = 10
print(-x)
print(not (x == 10))
```
**Output:**
```text
-10
False
```
