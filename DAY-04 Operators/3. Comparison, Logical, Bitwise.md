## 3. Comparison (Relational) Operators

Comparison operators are used to compare two values. They **always return a Boolean value** (`True` or `False`). They are the core of all `if` statements and `while` loops!

### A. Equal (`==`) and Not Equal (`!=`)
- `==` checks if the value on the left is exactly equal to the value on the right.
- `!=` checks if the value on the left is NOT equal to the value on the right.

**Input:**
```python
x = 10
y = 10
z = 5

print("Is x == y?", x == y)
print("Is x == z?", x == z)
print("Is x != z?", x != z)
```
**Output:**
```text
Is x == y? True
Is x == z? False
Is x != z? True
```

### B. Greater Than (`>`) and Less Than (`<`)
- `>` checks if the left value is strictly larger than the right value.
- `<` checks if the left value is strictly smaller than the right value.

**Input:**
```python
a = 20
b = 15

print("Is a > b?", a > b)
print("Is a < b?", a < b)
```
**Output:**
```text
Is a > b? True
Is a < b? False
```

### C. Greater Than or Equal (`>=`) and Less Than or Equal (`<=`)
- `>=` returns `True` if the left value is greater than OR exactly equal to the right.
- `<=` returns `True` if the left value is less than OR exactly equal to the right.

**Input:**
```python
score = 85
passing_grade = 85

# 85 is not > 85, but it IS >= 85!
print("Did student pass?", score >= passing_grade) 
```
**Output:**
```text
Did student pass? True
```

## 4. Logical Operators

Logical operators are used to combine multiple conditional statements together.

### A. The `and` Operator
Returns `True` ONLY if **both** statements are true. If even one is false, the whole expression is false.

**Input:**
```python
temperature = 25
is_sunny = True

# Both conditions must be met to go to the beach
if temperature > 20 and is_sunny == True:
    print("Perfect weather for the beach!")
else:
    print("Maybe stay inside.")
```
**Output:**
```text
Perfect weather for the beach!
```

### B. The `or` Operator
Returns `True` if **at least one** of the statements is true.

**Input:**
```python
has_ticket = False
has_vip_pass = True

# As long as they have ONE of the required items, they can enter
if has_ticket or has_vip_pass:
    print("Welcome to the event!")
```
**Output:**
```text
Welcome to the event!
```

### C. The `not` Operator
Reverses the result of the condition. If the condition is True, `not` makes it False.

**Input:**
```python
is_raining = True

# If it is NOT raining, we go for a walk
if not is_raining:
    print("Going for a walk!")
else:
    print("Bringing an umbrella.")
```
**Output:**
```text
Bringing an umbrella.
```

## 5. Identity Operators (`is`, `is not`)

Identity operators are used to compare the memory locations of two objects, not just if their values are equal.

**Input:**
```python
list1 = ["apple", "banana"]
list2 = ["apple", "banana"]
list3 = list1 # list3 points to the exact same memory as list1

print("list1 == list2:", list1 == list2) # True, values are the same
print("list1 is list2:", list1 is list2) # False, they are different lists in memory!
print("list1 is list3:", list1 is list3) # True, they are the exact same object
```
**Output:**
```text
list1 == list2: True
list1 is list2: False
list1 is list3: True
```

---

## 6. Membership Operators (`in`, `not in`)

Membership operators test if a sequence (like a string, list, or tuple) is present within an object.

**Input:**
```python
vowels = "aeiou"

print("Is 'a' in vowels?", 'a' in vowels)
print("Is 'z' in vowels?", 'z' in vowels)

shopping_list = ["milk", "bread", "eggs"]
print("Is 'cheese' NOT in shopping list?", "cheese" not in shopping_list)
```
**Output:**
```text
Is 'a' in vowels? True
Is 'z' in vowels? False
Is 'cheese' NOT in shopping list? True
```

## 7. Bitwise Operators

Bitwise operators are used to compare (binary) numbers. They operate at the bit level.

| Operator | Name | Description | Example |
|---|---|---|---|
| `&` | AND | Sets each bit to 1 if both bits are 1 | `x & y` |
| `\|` | OR | Sets each bit to 1 if one of two bits is 1 | `x \| y` |
| `^` | XOR | Sets each bit to 1 if only one of two bits is 1 | `x ^ y` |
| `~` | NOT | Inverts all the bits | `~x` |
| `<<` | Zero fill left shift | Shift left by pushing zeros in from the right | `x << 2` |
| `>>` | Signed right shift | Shift right by pushing copies of the leftmost bit in | `x >> 2` |

**Input:**
```python
# 6 in binary is 0110
# 3 in binary is 0011

print("Bitwise AND (6 & 3):", 6 & 3) # Output: 0010 (2)
print("Bitwise OR (6 | 3):", 6 | 3)  # Output: 0111 (7)
```
**Output:**
```text
Bitwise AND (6 & 3): 2
Bitwise OR (6 | 3): 7
```
