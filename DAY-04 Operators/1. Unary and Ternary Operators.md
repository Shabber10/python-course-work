# 1. Unary and Ternary Operators

Operators in Python are special symbols that perform operations on variables and values. In this section, we cover two unique categories of operators: **Unary Operators** (which operate on a single value) and **Ternary Operators** (which operate on three values, providing a quick conditional shortcut).

---

## 1. Unary Operators
A unary operator is an operator that operates on only **one** operand. 

### A. Unary Plus (`+`) and Minus (`-`)
The unary plus indicates a positive value (though it's usually omitted since numbers are positive by default). The unary minus mathematically negates a value, turning positive numbers negative, and negative numbers positive.

**Input:**
```python
x = 10
y = -50

# Using Unary Minus
print(-x)
print(-y)
```
**Output:**
```text
-10
50
```

### B. Logical NOT (`not`)
The `not` operator is a logical unary operator. It reverses the boolean state of its operand. If a condition is `True`, `not` makes it `False`.

**Input:**
```python
is_raining = True
print("Is it NOT raining?", not is_raining)

# It can also be used in logical expressions
age = 15
print("Is age not 18?", not (age == 18))
```
**Output:**
```text
Is it NOT raining? False
Is age not 18? True
```

### C. Bitwise NOT (`~`)
The bitwise NOT operator inverts all the bits in the binary representation of a number. Specifically, for any integer `x`, `~x` is the same as `-(x + 1)`.

**Input:**
```python
a = 5
print(~a)
```
**Output:**
```text
-6
```

---

## 2. Ternary Operator (Conditional Expressions)

The Ternary Operator is a fantastic tool in Python that allows you to write an entire `if-else` statement on a single line! It evaluates a condition and returns a value based on whether the condition is true or false.

**Syntax:** `[value_if_true] if [condition] else [value_if_false]`

### A. Simple Assignment
Instead of writing 4 lines of code for an if-else block to assign a value, you can do it in 1 line.

**Input:**
```python
age = 20

# Standard if-else assignment
if age >= 18:
    status = "Adult"
else:
    status = "Minor"

# Ternary Operator Assignment (The exact same logic!)
status_ternary = "Adult" if age >= 18 else "Minor"

print(status)
print(status_ternary)
```
**Output:**
```text
Adult
Adult
```

### B. Inline Print Statements
You can directly print the result of a ternary operator.

**Input:**
```python
score = 85

print("Pass!" if score >= 50 else "Fail!")
```
**Output:**
```text
Pass!
```

### C. Why Use the Ternary Operator?
- **Readability:** It makes code much cleaner and easier to read for simple assignments.
- **Brevity:** It saves vertical space in your scripts.
*Note: Do not use ternary operators for highly complex nested conditions, as it becomes harder to read than a standard multi-line `if-else` block.*
