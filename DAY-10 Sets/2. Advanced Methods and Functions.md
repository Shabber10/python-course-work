## 2. Deep Dive: Advanced Set Methods

### `isdisjoint()`, `issubset()`, and `issuperset()`
These return Booleans (`True`/`False`) based on the relationship between two sets.
- **Disjoint:** True if the two sets have absolutely ZERO items in common.
- **Subset:** True if all items in set x are present in set y.
- **Superset:** True if all items in set y are present in set x.

**Input:**
```python
x = {"a", "b", "c"}
y = {"f", "e", "d", "c", "b", "a"}
z = {"x", "y", "z"}

print("Are x and z disjoint?", x.isdisjoint(z))
print("Is x a subset of y?", x.issubset(y))
print("Is y a superset of x?", y.issuperset(x))
```

**Output:**
```text
Are x and z disjoint? True
Is x a subset of y? True
Is y a superset of x? True
```


### `symmetric_difference()`
Unlike standard difference (which only checks one way), symmetric difference returns a new set containing ALL items from BOTH sets, *except* the items that are present in both (it returns the non-overlap!).

**Input:**
```python
x = {"apple", "banana", "cherry"}
y = {"google", "microsoft", "apple"}

z = x.symmetric_difference(y)
print("Symmetric Difference:", z)
```

**Output:**
```text
Symmetric Difference: {'google', 'banana', 'cherry', 'microsoft'}
```


### The `_update()` Methods
Methods like `difference_update()`, `intersection_update()`, and `symmetric_difference_update()` perform the math operation, but instead of returning a new set, they **permanently overwrite the original set**.

**Input:**
```python
x = {"apple", "banana", "cherry"}
y = {"google", "microsoft", "apple"}

# Overwrites x with the intersection!
x.intersection_update(y)
print(x)
```

**Output:**
```text
{'apple'}
```

## 3. Deep Dive: Built-in Functions

### `min()`, `max()`, `sum()`
**Input:**
```python
price_set = {10, 50, 25, 5, 100}

print("Highest price:", max(price_set))
print("Lowest price:", min(price_set))
print("Sum of all prices:", sum(price_set))
```

**Output:**
```text
Highest price: 100
Lowest price: 5
Sum of all prices: 190
```


### The `set()` Constructor
You can instantly remove all duplicates from a List by passing it through the `set()` constructor function! This is one of the most common tricks in all of Python.

**Input:**
```python
# We have a list with tons of duplicate names!
names_list = ["Alice", "Bob", "Alice", "Charlie", "Bob", "Alice"]

# Convert the list into a set. Sets DESTROY duplicates automatically!
unique_names_set = set(names_list)

print("Original List:", names_list)
print("Unique Set:", unique_names_set)
```

**Output:**
```text
Original List: ['Alice', 'Bob', 'Alice', 'Charlie', 'Bob', 'Alice']
Unique Set: {'Bob', 'Alice', 'Charlie'}
```
