# 3. Escape Characters and Boolean Methods

When working with strings, you will eventually need to insert illegal characters or test the contents of the string to see if it contains numbers, letters, or spaces.

---

## 1. Escape Characters

To insert characters that are illegal in a string, use an **escape character**. An escape character is a backslash `\` followed by the character you want to insert.

For example, if you want to use double quotes *inside* a string that is surrounded by double quotes, Python will get confused and throw an error!

**Input:**
```python
# txt = "We are the so-called "Vikings" from the north." # ERROR!
```

To fix this, we escape the inner quotes using `\"`:

**Input:**
```python
txt = "We are the so-called \"Vikings\" from the north."
print(txt)
```
**Output:**
```text
We are the so-called "Vikings" from the north.
```

### Common Escape Characters
Here is a table of the most common escape characters used in Python:

| Code | Result | Description |
|---|---|---|
| `\'` | Single Quote | Escapes a single quote |
| `\"` | Double Quote | Escapes a double quote |
| `\\` | Backslash | Escapes a backslash so it prints normally |
| `\n` | New Line | Forces the text to drop to the next line |
| `\t` | Tab | Adds a large tab space |

**Input:**
```python
print("Line One\nLine Two") # New Line
print("Name:\tJohn Doe")     # Tab
print("This is a backslash: \\") # Printing a backslash
```
**Output:**
```text
Line One
Line Two
Name:	John Doe
This is a backslash: \
```

## 2. Boolean String Methods

Python provides several built-in methods that return a Boolean (`True` or `False`) based on the contents of the string. These are crucial for validating user input!

### `isalpha()`
Returns `True` if **all** the characters in the text are letters (A-Z).
**Input:**
```python
print("CompanyX".isalpha())   # False, contains an X but wait, CompanyX is all letters! True!
print("Company123".isalpha()) # False, contains numbers
print("Company X".isalpha())  # False, contains a space
```
**Output:**
```text
True
False
False
```

### `isdigit()` and `isnumeric()`
Returns `True` if **all** the characters are numbers (0-9). This is extremely helpful before trying to cast a string to an integer!
**Input:**
```python
print("50800".isdigit())      # True
print("50.8".isdigit())       # False, contains a decimal point
print("Hello".isdigit())      # False
```
**Output:**
```text
True
False
False
```

### `isspace()`
Returns `True` if **all** the characters in the text are whitespaces.
**Input:**
```python
print("   ".isspace())   # True
print("   a   ".isspace()) # False
```
**Output:**
```text
True
False
```

### `startswith()` and `endswith()`
Checks if the string starts or ends with a specific value.
**Input:**
```python
txt = "Hello, welcome to my world."
print(txt.startswith("Hello")) # True
print(txt.endswith("."))       # True
print(txt.endswith("world"))   # False, it ends with a period!
```
**Output:**
```text
True
True
False
```
