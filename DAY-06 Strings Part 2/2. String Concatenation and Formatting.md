# String Concatenation and Formatting

Often, you will need to combine multiple strings together, or insert variables directly into a string. Python provides several ways to do this, ranging from basic addition to advanced formatting techniques.

---

## 6. String Concatenation

To combine two or more strings, you can use the `+` operator. This process is called **Concatenation**.

**Input:**
```python
a = "Hello"
b = "World"
c = a + b

print(c)
```
**Output:**
```text
HelloWorld
```

### Adding Spaces
Notice that the `+` operator does NOT automatically add a space between the words. If you want a space, you must manually concatenate a space string `" "`.

**Input:**
```python
a = "Hello"
b = "World"
c = a + " " + b

print(c)
```
**Output:**
```text
Hello World
```

---

## 7. String Formatting (The `.format()` Method)

We CANNOT combine strings and numbers together using the `+` operator. Python will throw a `TypeError`!

**Input:**
```python
age = 36
# txt = "My name is John, I am " + age  # THIS WILL CRASH!
```

To combine strings and numbers, we use the `format()` method! The `format()` method takes the passed arguments, formats them, and places them in the string where the placeholders `{}` are.

**Input:**
```python
age = 36
txt = "My name is John, and I am {} years old."
print(txt.format(age))
```
**Output:**
```text
My name is John, and I am 36 years old.
```

### Multiple Placeholders
The `format()` method can take an unlimited number of arguments, and they are placed into the respective placeholders sequentially.

**Input:**
```python
quantity = 3
item_no = 567
price = 49.95
my_order = "I want {} pieces of item {} for {} dollars."

print(my_order.format(quantity, item_no, price))
```
**Output:**
```text
I want 3 pieces of item 567 for 49.95 dollars.
```

### Indexed Placeholders
You can use index numbers `{0}` to be sure the arguments are placed in the correct placeholders, even if you want to use the same argument multiple times!

**Input:**
```python
quantity = 3
item_no = 567
price = 49.95
my_order = "I want to pay {2} dollars for {0} pieces of item {1}."

print(my_order.format(quantity, item_no, price))
```
**Output:**
```text
I want to pay 49.95 dollars for 3 pieces of item 567.
```

## 8. f-Strings (Python 3.6+)

As of Python version 3.6, the absolute best, most modern way to format strings is using **f-Strings**. 

By putting an `f` before the first quote, you can put variables *directly inside* the curly braces `{}` without needing to use `.format()`. It is incredibly clean and readable!

**Input:**
```python
name = "Alice"
age = 25
score = 98.5

# Notice the 'f' at the very beginning of the string!
profile = f"Student Name: {name}, Age: {age}, Score: {score}"
print(profile)

# You can even do math directly inside the curly braces!
print(f"In 5 years, {name} will be {age + 5} years old.")
```
**Output:**
```text
Student Name: Alice, Age: 25, Score: 98.5
In 5 years, Alice will be 30 years old.
```

## 9. Joining Strings (`.join()`)

While `+` is used to join two string variables, what if you have a massive List of 100 strings that you want to combine into a single string? You use the `.join()` method!

The `.join()` method takes all items in an iterable (like a List or Tuple) and joins them into one string using a specified separator.

**Input:**
```python
# A list of words
words = ["Python", "is", "a", "great", "language"]

# Join them with a space
sentence = " ".join(words)
print("With Spaces:", sentence)

# Join them with a dash
dashed = "-".join(words)
print("With Dashes:", dashed)
```
**Output:**
```text
With Spaces: Python is a great language
With Dashes: Python-is-a-great-language
```
