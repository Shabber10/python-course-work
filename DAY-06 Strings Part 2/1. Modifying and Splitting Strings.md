# 1. Modifying and Splitting Strings

Python has a set of powerful built-in methods that you can use on strings to format, modify, and split them without needing to use complex loops or external libraries.

---

## 1. Changing Case (Upper and Lower)

You can instantly change the entire casing of a string using `.upper()` and `.lower()`.

**Input:**
```python
a = "Hello, World!"
print("Original:", a)

print("Uppercase:", a.upper())
print("Lowercase:", a.lower())
```
**Output:**
```text
Original: Hello, World!
Uppercase: HELLO, WORLD!
Lowercase: hello, world!
```

---

## 2. Removing Whitespace (Trimming)

Whitespace is the space before and/or after the actual text. When taking input from users, they often accidentally type extra spaces. We use the `.strip()` method to remove any whitespace from the beginning or the end.

**Input:**
```python
a = "   Hello, World!   "
print("Before strip: '" + a + "'")
print("After strip: '" + a.strip() + "'")
```
**Output:**
```text
Before strip: '   Hello, World!   '
After strip: 'Hello, World!'
```

---

## 3. Replacing Text

The `.replace()` method replaces a string with another string. This is incredibly useful for cleaning up data!

**Input:**
```python
a = "Hello, World!"

# Replace 'H' with 'J'
print(a.replace("H", "J"))

# Replace 'World' with 'Universe'
print(a.replace("World", "Universe"))
```
**Output:**
```text
Jello, World!
Hello, Universe!
```

---

## 4. Splitting Strings into Lists

The `.split()` method splits a single string into a **List of strings** based on a specified separator (like a comma, or a space). 

This is the absolute most common way to process comma-separated values (CSV) data in Python!

**Input:**
```python
# Splitting by a comma
a = "apple, banana, cherry"
fruits_list = a.split(",")

print(fruits_list)
print("The first fruit is:", fruits_list[0])

# Splitting by a space (the default behavior)
sentence = "Python is a great language"
words_list = sentence.split(" ")

print(words_list)
```
**Output:**
```text
['apple', ' banana', ' cherry']
The first fruit is: apple
['Python', 'is', 'a', 'great', 'language']
```
