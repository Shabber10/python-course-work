# Type Conversion and Casting

There may be times when you want to specify a type on to a variable. This can be done with casting. Python is an object-orientated language, and as such it uses classes to define data types, including its primitive types.

Casting in python is therefore done using constructor functions:

---

## 1. Explicit Type Casting

### `int()`
Constructs an integer number from an integer literal, a float literal (by removing all decimals), or a string literal (providing the string represents a whole number).
**Input:**
```python
x = int(1)   # x will be 1
y = int(2.8) # y will be 2
z = int("3") # z will be 3
print(x, y, z)
```
**Output:**
```text
1 2 3
```

### `float()`
Constructs a float number from an integer literal, a float literal, or a string literal (providing the string represents a float or an integer).
**Input:**
```python
x = float(1)     # x will be 1.0
y = float(2.8)   # y will be 2.8
z = float("3")   # z will be 3.0
w = float("4.2") # w will be 4.2
print(x, y, z, w)
```
**Output:**
```text
1.0 2.8 3.0 4.2
```

### `str()`
Constructs a string from a wide variety of data types, including strings, integer literals, and float literals.
**Input:**
```python
x = str("s1") # x will be 's1'
y = str(2)    # y will be '2'
z = str(3.0)  # z will be '3.0'
print(x, y, z)
```
**Output:**
```text
s1 2 3.0
```

### Casting Quirks & Handling Errors

1. **Negative Numbers:** Casting works with signed negative numbers as well.
   - `int("-10") -> -10`
   - `float("-3.14") -> -3.14`
2. **Invalid Conversions:** Converting a non-numeric string (e.g., `"hello"`) to a number raises a `ValueError`.
3. **Casting User Input:** The `input()` function always returns a string. To perform arithmetic on numerical input, you must explicitly wrap it in `int()` or `float()`.

**Input:**
```python
# 1. Invalid conversion check
try:
    int("hello")
except ValueError as e:
    print("Error:", e)

# 2. Arithmetic on input conversion
score_input = "50"
final_score = int(score_input) + 10
print("Final Score:", final_score)
```
**Output:**
```text
Error: invalid literal for int() with base 10: 'hello'
Final Score: 60
```
