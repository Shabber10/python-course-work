## 2. Getting the Data Type (`type()`)
You can get the data type of any object by using the `type()` function.

**Input:**
```python
x = 10        # int
y = 3.14      # float
z = 1j        # complex
name = "Bob"  # str
is_active = True # bool

print(type(x))
print(type(z))
print(type(is_active))
```
**Output:**
```text
<class 'int'>
<class 'complex'>
<class 'bool'>
```
