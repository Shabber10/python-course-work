# 1. Built-in Data Types Deep Dive

In programming, a **data type** is an important concept. Variables can store data of different types, and different types can do different things. Python is completely object-oriented, and not "statically typed". You do not need to declare variables with any particular type, and you can even change their type after they have been set!

Python has the following standard or built-in data types grouped into categories:

---

## 1. Getting the Data Type with `type()`
You can instantly check the data type of any object by using the built-in `type()` function. This is incredibly useful for debugging.

**Input:**
```python
x = 5
y = "Hello World"
is_active = True

print(type(x))
print(type(y))
print(type(is_active))
```
**Output:**
```text
<class 'int'>
<class 'str'>
<class 'bool'>
```

---

## 2. Numeric Types (`int`, `float`, `complex`)

Python handles numbers flawlessly. There are three distinct numeric types:

### A. Int (Integer)
Int, or integer, is a whole number, positive or negative, without decimals, of unlimited length.
**Input:**
```python
x = 1
y = 35656222554887711
z = -3255522

print(type(x))
print(type(y))
print(type(z))
```
**Output:**
```text
<class 'int'>
<class 'int'>
<class 'int'>
```

### B. Float (Floating Point)
Float is a number, positive or negative, containing one or more decimals. It can also be scientific numbers with an "e" to indicate the power of 10.
**Input:**
```python
x = 1.10
y = -35.59
z = 35e3  # 35 * 10^3

print(type(x))
print(type(y))
print(z)
print(type(z))
```
**Output:**
```text
<class 'float'>
<class 'float'>
35000.0
<class 'float'>
```

### C. Complex
Complex numbers are written with a "j" as the imaginary part. They are heavily used in scientific and engineering calculations.
**Input:**
```python
x = 3+5j
y = 5j
z = -5j

print(type(x))
print(type(y))
print(type(z))
```
**Output:**
```text
<class 'complex'>
<class 'complex'>
<class 'complex'>
```

---

## 3. Text Type (`str`)
Strings in Python are surrounded by either single quotation marks, or double quotation marks. They are actually arrays of bytes representing Unicode characters, allowing you to slice and dice them easily.

**Input:**
```python
a = "Hello"
b = 'World'
c = """This is a
multiline string."""

print(type(a))
print(a)
print(c)
```
**Output:**
```text
<class 'str'>
Hello
This is a
multiline string.
```

---

## 4. Boolean Type (`bool`)
Booleans represent one of two values: `True` or `False`. They are vital for conditional testing.

**Input:**
```python
print(10 > 9)
print(10 == 9)
print(10 < 9)
```
**Output:**
```text
True
False
False
```

---

## 5. Sequence, Mapping, and Set Types
Python provides powerful advanced data structures to hold collections of data.

1. **Sequence Types (`list`, `tuple`, `range`)**: Ordered collections. Lists are mutable, tuples are immutable.
2. **Mapping Type (`dict`)**: Key-value pairs. Think of a real-world dictionary.
3. **Set Types (`set`, `frozenset`)**: Unordered collections of *unique* items.
