# Built-in Data Types

In programming, a **data type** is one of the most fundamental concepts. Variables can store data of different types, and different types can do different things. 

Python is completely object-oriented, and not "statically typed". You do not need to declare variables with any particular type, and you can even change their type after they have been set simply by assigning a new value!

---

## 8 Types of Data Types
Python has the following standard or built-in data types grouped into categories:

| Category | Data Types |
| :--- | :--- |
| **Text Type:** | `str` |
| **Numeric Types:** | `int`, `float`, `complex` |
| **Sequence Types:** | `list`, `tuple`, `range` |
| **Mapping Type:** | `dict` |
| **Set Types:** | `set`, `frozenset` |
| **Boolean Type:** | `bool` |
| **Binary Types:** | `bytes`, `bytearray`, `memoryview` |
| **None Type:** | `NoneType` |

---

## 1. Numeric Types (`int`, `float`, `complex`)

### Int (Integer)
Int, or integer, is a whole number, positive or negative, without decimals, of unlimited length.
**Input:**
```python
b = 20              # int
y = 35656222554887711
z = -3255522
```

### Float (Floating Point)
Float is a number, positive or negative, containing one or more decimals. 
**Input:**
```python
c = 20.5            # float
y = -35.59
z = 35e3            # 35 * 10^3
```

### Complex
Complex numbers are written with a "j" as the imaginary part.
**Input:**
```python
d = 1j              # complex
x = 3+5j
```

---

## 2. Text Type (`str`)
Strings in Python are surrounded by either single quotation marks, or double quotation marks. 
**Input:**
```python
a = "Hello World"   # str
b = 'World'
```

---

## 3. Boolean Type (`bool`)
Booleans represent one of two values: `True` or `False`. 
**Input:**
```python
j = True            # bool
k = False
```

---

## 4. None Type (`NoneType`)
`None` is used to define a null variable or an object. In Python, `None` is an object of its own datatype.
**Input:**
```python
k = None            # NoneType
```

---

## 5. Binary Types (`bytes`, `bytearray`, `memoryview`)
Binary types are advanced objects used to manipulate raw binary data.
**Input:**
```python
l = b"Hello"        # bytes
m = bytearray(5)    # bytearray
n = memoryview(bytes(5)) # memoryview
```
