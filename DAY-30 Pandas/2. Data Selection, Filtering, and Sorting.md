# 2. Data Selection, Filtering, and Sorting

Selecting, filtering, and reorganizing data are fundamental steps in any data analysis workflow. Pandas provides highly optimized methods to subset and query DataFrames.

---

## 1. Column Selection

You can select columns from a DataFrame using bracket notation or attribute access:
- **`df["ColumnName"]`**: Returns a **Series** (1D).
- **`df[["Col1", "Col2"]]`**: Returns a subset **DataFrame** (2D). Note the double brackets.

**Input:**
```python
import pandas as pd

df = pd.DataFrame({
    "Name": ["Alice", "Bob", "Charlie"],
    "Age": [25, 30, 35],
    "Salary": [50000, 60000, 70000]
})

# 1. Select a single column (returns a Series)
names = df["Name"]
print("Single Column (Type:", type(names), "):")
print(names)

# 2. Select multiple columns (returns a DataFrame)
subset = df[["Name", "Salary"]]
print("\nMultiple Columns (Type:", type(subset), "):")
print(subset)
```
**Output:**
```text
Single Column (Type: <class 'pandas.core.series.Series'> ):
0      Alice
1        Bob
2    Charlie
Name: Name, dtype: object

Multiple Columns (Type: <class 'pandas.core.frame.DataFrame'> ):
      Name  Salary
0    Alice   50000
1      Bob   60000
2  Charlie   70000
```

## 2. Row & Element Selection: `.loc` vs. `.iloc`

Pandas provides two primary indexers to select rows and columns simultaneously:

| Indexer | Selection Basis | Syntax |
|:---|:---|:---|
| **`.loc`** | **Labels** (names of rows and columns). | `df.loc[row_label, col_label]` |
| **`.iloc`** | **Integer Positions** (0-indexed indices). | `df.iloc[row_pos, col_pos]` |

```text
       DataFrame Coordinates:
  
                Col Position 0      Col Position 1
                Col Label 'Name'    Col Label 'Age'
              ┌───────────────────┬─────────────────┐
  Row Pos 0   │                   │                 │
  Row Label 0 │      Alice        │       25        │
              ├───────────────────┼─────────────────┤
  Row Pos 1   │                   │                 │
  Row Label 1 │       Bob         │       30        │
              └───────────────────┴─────────────────┘
```

**Input:**
```python
import pandas as pd

# Creating DataFrame with a custom index label ('a', 'b', 'c')
df = pd.DataFrame({
    "Name": ["Alice", "Bob", "Charlie"],
    "Age": [25, 30, 35]
}, index=["a", "b", "c"])

print("--- DataFrame ---")
print(df)

# 1. Label-based selection (.loc)
print("\nRow 'b':\n", df.loc["b"])
print("Value at row 'b', column 'Name':", df.loc["b", "Name"])

# 2. Position-based selection (.iloc)
print("\nRow at index position 1:\n", df.iloc[1])
print("Value at Row 1, Col 0:", df.iloc[1, 0])

# 3. Slicing with loc/iloc
print("\nSlicing first 2 rows using .iloc:\n", df.iloc[0:2])
```
**Output:**
```text
--- DataFrame ---
      Name  Age
a    Alice   25
b      Bob   30
c  Charlie   35

Row 'b':
 Name    Bob
Age      30
Name: b, dtype: object
Value at row 'b', column 'Name': Bob

Row at index position 1:
 Name    Bob
Age      30
Name: b, dtype: object
Value at Row 1, Col 0: Bob

Slicing first 2 rows using .iloc:
     Name  Age
a  Alice   25
b    Bob   30
```

## 3. Conditional Filtering (Boolean Indexing)

To filter rows based on values within columns, pass a boolean condition inside brackets.

### Logical Operators:
- **`&`** (AND) - both conditions must be True.
- **`|`** (OR) - at least one condition must be True.
- **`~`** (NOT) - negates a condition.

> [!IMPORTANT]
> **Parentheses Rule:**
> When combining multiple conditions in Pandas, you **must** wrap each condition in parentheses `()`. Standard Python `and`, `or`, and `not` words will raise an exception.

**Input:**
```python
import pandas as pd

df = pd.DataFrame({
    "Name": ["Alice", "Bob", "Charlie", "David"],
    "Age": [25, 30, 35, 40],
    "Dept": ["HR", "IT", "IT", "Finance"]
})

# 1. Simple condition: Age >= 30
print("--- Age >= 30 ---")
print(df[df["Age"] >= 30])

# 2. Multiple conditions (AND): IT department AND Age > 30
print("\n--- IT Dept AND Age > 30 ---")
print(df[(df["Dept"] == "IT") & (df["Age"] > 30)])

# 3. Filtering by a list of values (.isin)
print("\n--- Dept in ['HR', 'Finance'] ---")
print(df[df["Dept"].isin(["HR", "Finance"])])
```
**Output:**
```text
--- Age >= 30 ---
      Name  Age     Dept
1      Bob   30       IT
2  Charlie   35       IT
3    David   40  Finance

--- IT Dept AND Age > 30 ---
      Name  Age Dept
2  Charlie   35   IT

--- Dept in ['HR', 'Finance'] ---
    Name  Age     Dept
0  Alice   25       HR
3  David   40  Finance
```

## 4. Sorting Data

Pandas allows sorting DataFrames by column values or by their row index.

- **`df.sort_values(by, ascending)`**: Sorts by one or more columns.
- **`df.sort_index(ascending)`**: Sorts by row index.

**Input:**
```python
import pandas as pd

df = pd.DataFrame({
    "Name": ["Alice", "Bob", "Charlie"],
    "Age": [35, 25, 30]
})

# 1. Sort by Age (ascending order)
print("--- Sorted by Age (Ascending) ---")
print(df.sort_values(by="Age"))

# 2. Sort by Age (descending order)
print("\n--- Sorted by Age (Descending) ---")
print(df.sort_values(by="Age", ascending=False))
```
**Output:**
```text
--- Sorted by Age (Ascending) ---
      Name  Age
1      Bob   25
2  Charlie   30
0    Alice   35

--- Sorted by Age (Descending) ---
      Name  Age
0    Alice   35
2  Charlie   30
1      Bob   25
```
