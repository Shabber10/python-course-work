# 4. Data Aggregation, Grouping, and Merging

When analyzing complex datasets, we frequently need to aggregate numbers by groups (e.g., finding the average salary per department) or combine multiple tables of data together (similar to SQL operations). Pandas makes these operations highly intuitive.

---

## 1. Grouping and Aggregating Data (`groupby`)

The **`groupby()`** process involves three steps:
1. **Splitting** the data into groups based on some keys.
2. **Applying** a function (like mean, sum, count) to each group.
3. **Combining** the results into a new DataFrame.

```text
                 SPLIT-APPLY-COMBINE FLOW:
  
     Original DataFrame           Splitting         Mean Applied        Combined
    ┌──────────┬───────┐        ┌───────────┐       ┌───────────┐      ┌──────────┬──────┐
    │   Dept   │ Sales │        │ HR Group  │       │ HR Avg    │      │   Dept   │Sales │
    ├──────────┼───────┤        │ (100, 200)│ ───>  │ (150)     │      ├──────────┼──────┤
    │    HR    │  100  │        └───────────┘       └───────────┘      │    HR    │ 150  │
    │    IT    │  300  │ ───>   ┌───────────┐       ┌───────────┐ ───> ├──────────┼──────┤
    │    HR    │  200  │        │ IT Group  │       │ IT Avg    │      │    IT    │ 300  │
    │    IT    │  300  │        │ (300, 300)│       │ (300)     │      └──────────┴──────┘
    └──────────┴───────┘        └───────────┘       └───────────┘
```

**Input:**
```python
import pandas as pd

df = pd.DataFrame({
    "Dept": ["HR", "IT", "Sales", "HR", "Sales", "IT"],
    "Salary": [50000, 70000, 60000, 52000, 62000, 80000],
    "Experience": [2, 5, 3, 3, 4, 7]
})

# 1. Group by Dept and find the mean Salary
avg_salary = df.groupby("Dept")["Salary"].mean()
print("--- Average Salary by Department ---")
print(avg_salary)

# 2. Applying multiple aggregates using .agg()
custom_agg = df.groupby("Dept").agg({
    "Salary": ["mean", "max"],
    "Experience": "sum"
})
print("\n--- Custom Aggregates ---")
print(custom_agg)
```
**Output:**
```text
--- Average Salary by Department ---
Dept
HR       51000.0
IT       75000.0
Sales    61000.0
Name: Salary, dtype: float64

--- Custom Aggregates ---
        Salary         Experience
          mean    max         sum
Dept                             
HR     51000.0  52000           5
IT     75000.0  80000          12
Sales  61000.0  62000           7
```

## 2. Concatenating DataFrames

Concatenation is the process of appending dataframes to each other (either row-wise or column-wise) using **`pd.concat([df1, df2], axis)`**:
- **`axis=0`** (default): Stacks them vertically on top of each other (adds rows).
- **`axis=1`**: Places them side-by-side (adds columns).

**Input:**
```python
import pandas as pd

df1 = pd.DataFrame({
    "Name": ["Alice", "Bob"],
    "Score": [85, 90]
})

df2 = pd.DataFrame({
    "Name": ["Charlie", "David"],
    "Score": [88, 92]
})

# Concatenating along axis=0 (stacking rows)
stacked_df = pd.concat([df1, df2], axis=0, ignore_index=True)
print("--- Vertically Stacked ---")
print(stacked_df)
```
**Output:**
```text
--- Vertically Stacked ---
      Name  Score
0    Alice     85
1      Bob     90
2  Charlie     88
3    David     92
```

## 3. Merging DataFrames (SQL-like Joins)

Merging combines DataFrames based on common key columns, similar to an SQL JOIN operation:
`pd.merge(left_df, right_df, on="common_column", how="join_type")`

### Join Types (`how`):
- **`'inner'`** (default): Keeps only rows where the key exists in both DataFrames.
- **`'left'`**: Keeps all rows from the left DataFrame, matching rows from the right.
- **`'right'`**: Keeps all rows from the right DataFrame, matching rows from the left.
- **`'outer'`**: Keeps all rows from both DataFrames, filling missing matches with NaN.

**Input:**
```python
import pandas as pd

# Table 1: Employees and their department IDs
employees = pd.DataFrame({
    "EmpID": [101, 102, 103, 104],
    "Name": ["Alice", "Bob", "Charlie", "David"]
})

# Table 2: Departments and their names
departments = pd.DataFrame({
    "EmpID": [101, 102, 103, 105],  # 104 is missing, 105 is new
    "Dept": ["HR", "IT", "Sales", "Finance"]
})

# 1. Inner Join (only matching keys)
inner_merge = pd.merge(employees, departments, on="EmpID", how="inner")
print("--- Inner Join ---")
print(inner_merge)

# 2. Left Join (keeps all employees, missing department is NaN)
left_merge = pd.merge(employees, departments, on="EmpID", how="left")
print("\n--- Left Join ---")
print(left_merge)
```
**Output:**
```text
--- Inner Join ---
   EmpID     Name   Dept
0    101    Alice     HR
1    102      Bob     IT
2    103  Charlie  Sales

--- Left Join ---
   EmpID     Name   Dept
0    101    Alice     HR
1    102      Bob     IT
2    103  Charlie  Sales
3    104    David    NaN
```
