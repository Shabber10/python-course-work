# 3. Handling Missing Data and Data Cleaning

In real-world data analysis, datasets are rarely clean. They often contain missing (null) values, duplicate records, incorrect formats, or typo-ridden column names. Data cleaning (wrangling) is one of the most vital stages of data science, and Pandas provides tools to handle these issues efficiently.

---

## 1. Detecting Missing Values

Pandas uses **`NaN`** (Not a Number) to represent missing numerical or textual data.

To check for missing values:
- **`isna()`** (or **`isnull()`**): Returns `True` for every cell containing missing data.
- **`isna().sum()`**: Summarizes the count of missing values in each column.

**Input:**
```python
import pandas as pd
import numpy as np

df = pd.DataFrame({
    "Name": ["Alice", "Bob", "Charlie", None],
    "Age": [25, np.nan, 35, 40],
    "Salary": [50000, 60000, np.nan, np.nan]
})

print("--- DataFrame ---")
print(df)

# Check missing cells
print("\n--- Cells with Missing Data (isna()) ---")
print(df.isna())

# Summarize missing data counts per column
print("\n--- Missing Data Count per Column ---")
print(df.isna().sum())
```
**Output:**
```text
--- DataFrame ---
      Name   Age   Salary
0    Alice  25.0  50000.0
1      Bob   NaN  60000.0
2  Charlie  35.0      NaN
3     None  40.0      NaN

--- Cells with Missing Data (isna()) ---
    Name    Age  Salary
0  False  False   False
1  False   True   False
2  False  False    True
3   True  False    True

--- Missing Data Count per Column ---
Name      1
Age       1
Salary    2
dtype: int64
```

---

## 2. Dropping Missing Values

You can drop missing values using **`dropna()`**:
- **`df.dropna()`**: Drops any row containing at least one missing value.
- **`df.dropna(axis=1)`**: Drops any column containing at least one missing value.
- **`df.dropna(subset=["ColName"])`**: Drops rows only if the specified columns have missing values.
- **`inplace=True`**: Modifies the original DataFrame instead of returning a new one.

**Input:**
```python
import pandas as pd
import numpy as np

df = pd.DataFrame({
    "Name": ["Alice", "Bob", "Charlie", "David"],
    "Age": [25, np.nan, 35, np.nan]
})

# Drop rows with ANY missing value
df_dropped_rows = df.dropna()
print("--- Dropped Rows ---")
print(df_dropped_rows)

# Drop rows where 'Age' is missing specifically
df_dropped_subset = df.dropna(subset=["Age"])
print("\n--- Dropped Rows via subset=['Age'] ---")
print(df_dropped_subset)
```
**Output:**
```text
--- Dropped Rows ---
    Name   Age
0  Alice  25.0
2  Charlie  35.0

--- Dropped Rows via subset=['Age'] ---
      Name   Age
0    Alice  25.0
2  Charlie  35.0
```

---

## 3. Filling Missing Values

Dropping rows can result in losing a huge amount of data. Instead, you can fill (impute) missing values using **`fillna()`**:
- Fill with a constant value.
- Fill with descriptive statistics (e.g., column mean or median).
- Fill using forward-fill (`ffill`) or backward-fill (`bfill`) to propagate values.

**Input:**
```python
import pandas as pd
import numpy as np

df = pd.DataFrame({
    "Name": ["Alice", "Bob", "Charlie", "David"],
    "Score": [90.0, np.nan, 80.0, np.nan]
})

# 1. Fill with a constant value (e.g., 0.0)
df_constant = df.copy()
df_constant["Score"] = df_constant["Score"].fillna(0.0)
print("--- Filled with 0.0 ---")
print(df_constant)

# 2. Fill with Mean/Average
df_mean = df.copy()
mean_score = df_mean["Score"].mean()
df_mean["Score"] = df_mean["Score"].fillna(mean_score)
print(f"\n--- Filled with Mean ({mean_score}) ---")
print(df_mean)
```
**Output:**
```text
--- Filled with 0.0 ---
      Name  Score
0    Alice   90.0
1      Bob    0.0
2  Charlie   80.0
3    David    0.0

--- Filled with Mean (85.0) ---
      Name  Score
0    Alice   90.0
1      Bob   85.0
2  Charlie   80.0
3    David   85.0
```

## 4. Handling Duplicates

Duplicate records skew statistical results and must be removed.

- **`df.duplicated()`**: Returns a Boolean Series indicating if a row is duplicate.
- **`df.drop_duplicates()`**: Removes duplicate rows.

**Input:**
```python
import pandas as pd

df = pd.DataFrame({
    "Name": ["Alice", "Bob", "Alice", "Alice"],
    "Age": [25, 30, 25, 25]
})

print("Check duplicated rows:\n", df.duplicated())

# Dropping duplicates
df_unique = df.drop_duplicates()
print("\nDataFrame after dropping duplicates:\n", df_unique)
```
**Output:**
```text
Check duplicated rows:
 0    False
1    False
2     True
3     True
dtype: bool

DataFrame after dropping duplicates:
     Name  Age
0  Alice   25
1    Bob   30
```

## 5. Renaming Columns & Replacing Values

- **`df.rename(columns={})`**: Renames column labels via a dictionary mapper.
- **`df.replace(to_replace, value)`**: Replaces specific values inside columns.

**Input:**
```python
import pandas as pd

df = pd.DataFrame({
    "emp_name": ["Alice", "Bob"],
    "status": ["Y", "N"]
})

# 1. Rename column headers
df = df.rename(columns={"emp_name": "EmployeeName", "status": "Active"})
print("--- Renamed Columns ---")
print(df)

# 2. Map Y/N status values to True/False
df["Active"] = df["Active"].replace({"Y": True, "N": False})
print("\n--- Replaced Values ---")
print(df)
```
**Output:**
```text
--- Renamed Columns ---
  EmployeeName Active
0        Alice      Y
1          Bob      N

--- Replaced Values ---
  EmployeeName  Active
0        Alice    True
1          Bob   False
```
