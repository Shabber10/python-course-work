# 1. Introduction to Pandas Series and DataFrames

While NumPy is excellent for handling homogeneous numerical data, real-world data is often tabular, containing diverse types of data (like strings, floats, booleans, and dates) alongside missing values. To handle this, we use **Pandas** (Panel Data)—the primary Python library for data analysis and manipulation.

---

## 1. What is Pandas and Why Use It?

Pandas builds on top of NumPy, offering two primary data structures:
1. **Series**: A 1-dimensional labeled array (like a single column of data).
2. **DataFrame**: A 2-dimensional tabular structure (like a spreadsheet or database table).

```text
               PANDAS DATA STRUCTURES:
  
        Series (1D)                   DataFrame (2D)
       ┌───────────┐           ┌───────────┬───────────┐
     0 │   Apple   │       Age │   Name    │   City    │
       ├───────────┤       ────┼───────────┼───────────┤
     1 │  Banana   │         0 │   John    │ New York  │
       ├───────────┤         1 │   Sarah   │  London   │
     2 │  Cherry   │         2 │   David   │   Paris   │
       └───────────┘           └───────────┴───────────┘
```

Before using Pandas, you must install it:
```bash
pip install pandas
```

In your scripts, it is imported by standard convention under the alias **`pd`**:
```python
import pandas as pd
```

---

## 2. Pandas Series

A **Series** is a 1-dimensional array that can hold any data type. Unlike a standard list or NumPy array, every element in a Series has an explicit, customizable index.

**Input:**
```python
import pandas as pd

# 1. Creating a Series from a List (default integer index)
fruits = pd.Series(["Apple", "Banana", "Cherry"])
print("--- Default Index Series ---")
print(fruits)

# 2. Creating a Series with Custom Indices
custom_fruits = pd.Series([10, 20, 30], index=["Apple", "Banana", "Cherry"])
print("\n--- Custom Index Series ---")
print(custom_fruits)

# 3. Accessing elements
print("\nAccessing by label 'Banana':", custom_fruits["Banana"])
print("Accessing by integer index 1:", custom_fruits.iloc[1])
```
**Output:**
```text
--- Default Index Series ---
0     Apple
1    Banana
2    Cherry
dtype: object

--- Custom Index Series ---
Apple     10
Banana    20
Cherry    30
dtype: int64

Accessing by label 'Banana': 20
Accessing by integer index 1: 20
```

## 3. Pandas DataFrames

A **DataFrame** is a table containing rows and columns. Columns have names, and rows have index labels.

### Creating DataFrames
You can construct DataFrames using dictionaries, lists of dictionaries, or NumPy arrays.

**Input:**
```python
import pandas as pd

# Creating a DataFrame from a dictionary of lists
data = {
    "Name": ["Alice", "Bob", "Charlie"],
    "Age": [25, 30, 22],
    "City": ["New York", "London", "Paris"]
}

df = pd.DataFrame(data)
print("--- DataFrame from Dictionary ---")
print(df)
```
**Output:**
```text
--- DataFrame from Dictionary ---
      Name  Age      City
0    Alice   25  New York
1      Bob   30    London
2  Charlie   22     Paris
```

## 4. Crucial DataFrame Attributes & Methods

Once you load a dataset (e.g., from a CSV file), the first step is to explore its shape, structure, and basic statistics.

### A. DataFrame Attributes
- **`.shape`**: Returns a tuple `(rows, columns)`.
- **`.columns`**: Returns a list/index of column names.
- **`.index`**: Returns the row index range/labels.
- **`.dtypes`**: Returns the data type of each column.

**Input:**
```python
import pandas as pd

df = pd.DataFrame({
    "Name": ["Alice", "Bob"],
    "Age": [25, 30]
})

print("Shape  :", df.shape)
print("Columns:", df.columns.tolist())
print("Index  :", df.index)
print("\nColumn Data Types:\n", df.dtypes)
```
**Output:**
```text
Shape  : (2, 2)
Columns: ['Name', 'Age']
Index  : RangeIndex(start=0, stop=2, step=1)

Column Data Types:
 Name    object
Age      int64
dtype: object
```

### B. Diagnostic & Summary Methods
- **`head(n)`**: Returns the first `n` rows of the DataFrame (default is 5).
- **`tail(n)`**: Returns the last `n` rows.
- **`info()`**: Prints a detailed summary of columns, count of non-null values, and memory usage.
- **`describe()`**: Generates descriptive statistics (mean, min, max, standard deviation) for numerical columns.

**Input:**
```python
import pandas as pd

data = {
    "Product": ["Laptop", "Mouse", "Keyboard", "Monitor", "Headphones"],
    "Price": [1200.0, 25.5, 75.0, 300.0, 80.0],
    "Stock": [5, 40, 25, 10, 15]
}
df = pd.DataFrame(data)

# 1. Previewing first 3 rows
print("--- First 3 Rows ---")
print(df.head(3))

# 2. DataFrame Information
print("\n--- DataFrame Information ---")
df.info()

# 3. Descriptive Statistics
print("\n--- Summary Statistics ---")
print(df.describe())
```
