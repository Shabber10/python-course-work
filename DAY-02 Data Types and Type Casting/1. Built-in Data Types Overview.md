## 1. Built-in Data Types Overview
- **Numeric Types:** `int`, `float`, `complex`
- **Sequence Types:** `str`, `list`, `tuple`
- **Mapping Type:** `dict`
- **Set Types:** `set`, `frozenset`
- **Boolean Type:** `bool`
- **Binary Types:** `bytes`, `bytearray`, `memoryview`
- **None Type:** `NoneType`
